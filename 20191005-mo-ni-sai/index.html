地下党

underground

题目背景

1949 年，解放前夕北平城内潜伏着许多地下党员。悠酱被任命为地下党指挥官，她希望计算地下党员之间的最小通信距离。

题目描述

我们不妨将北平城视为由$N$个点和$M$条边组成的无向图，$K$名地下党员分布在这些点上($K\leq N$)。

所谓最小通信距离，就是任意两名地下党员间最短路的最小值。

输入输出格式

输入格式

第一行一个整数$T$，代表数据组数，接下来一共$T$组数据。

每组数据第一行为三个整数，$N,M,K$，分别表示点数、边数、地下党员数。

之后$M$行，每行三个整数，$u,v,w$，表示$u$和$v$之间有一条长$w$的无向边。

之后$1$行，共$K$个整数，表示地下党员所在的结点编号。

输出格式

输出一共$T$行，每行一个整数，代表最小通信距离。

注意如果最小通信距离为$\infty$，请输出$-1$代替。

输入输出样例

输入样例 #1

输出样例 #1

说明

$N\leq 100000, M\leq 200000$ 且 $0\leq w\leq 1000$。

具体的看 pdf 吧。我实在拿纯 $\LaTeX$ 没办法。

题解

看着好像多源最短路诶。

其实不用。。。

发现能对答案造成贡献的，一定是不同的点（废话）。

不同的点呢，一定会有一个二进制位不同的。

于是就枚举到底是哪个二进制位不同，然后把这一位是 $1$（哪个都无所谓，都一样的）的一堆 $dis$ 设为 $0$，跑最短路，用这一位为 $0$ 的 $dis$ 更新答案。

任羽辰当时没看到题里是给开O2的，然后拿自己的机子跑标程花了7s多，于是就把测试时限也调到了7s多。

大集会

gathering

题目背景

2022年，悠酱、任酱和你都毕业了。

虽然各自奔向了不同的城市，但是距离并不能淡化你们的友谊。

你们约定每年都要选择一个城市相聚。

题目描述

世界上有$N$座城市，由$N-1$条双向航线连通。

每年，悠酱、任酱和你都可能在不同于去年的城市。

每年，悠酱、任酱和你都要选择一个城市相聚，你们希望三人到达该城市所经过的总航线数尽可能少。

希望你能挑选最合适的城市，保证最优的选择是唯一的。

输入输出格式

输入格式

第一行两个整数，$N,Q$，代表有$N$座城市，共$Q$年的询问。

之后$N-1$行，每行两个整数，$u,v$，表示城市$u$和$v$之间有一条双向航线。

之后$Q$行，每行三个整数，$a,b,c$，表示悠酱、任酱和你该年所在城市。

输出格式

共$Q$行，每行以空格分割的两个整数，$P,S$，表示该年最合适的聚会城市为$P$，三人经过的总航线数为$S$。

输入输出样例

输入样例 #1

输出样例 #1

说明

$N,Q\leq 5\times 10^5$

放射性

radioactive

题目背景

2077 年，地球资源日益枯竭。为了提供足够支撑超级计算机“树形图”运行的能量，悠酱被派遣到核泄漏区域收集放射性物质。

题目描述

悠酱观测到了$N$堆放射性物质。在第$0$时刻，第$i$堆放射性物质具有$E_i$单位能量。

不幸的是，这些放射性物质的能量正在不断向外流失。

具体而言，第$i$堆放射性物质每单位时间会流失$L_i$单位能量。

聪明的悠酱已经知道采集第$i$堆放射性物质需要$T_i$单位时间。

现在，我们假定悠酱不需要休息、移动不需要时间、采集第$i$堆放射性物质时其不会流失能量。

那么，悠酱最多可以采集到多少单位能量的放射性物质呢？

输入输出格式

输入格式

第一行一个整数$T$，代表数据组数，接下来一共$T$组数据。

每组数据的第一行为一个整数，$N$，代表放射性物质堆数。

接下来$N$行，每行三个整数，$E_i,L_i,T_i$，表示第$i$堆物质的属性。

输出格式

输出一共$T$行，每行一个整数，代表最多能采集到的能量。

输入输出样例

输入样例 #1

输出样例 #1

说明

$ N\leq 100, L_i\leq 100, T_i\leq 10000, E_i\leq 10000$

题解

这题可以先想怎么让损失最小（指流失可能会造成的 $E_i$ 减小，但是不考虑 $E_i$ 的具体值，那是后面背包的内容）。

当 $T_i=1$ 的时候，肯定要找流得快的先拿啊，这样能把损失降到最小。

但是 $T_i$ 不是 $1$ 了呢？想想拿 $i$ 在什么时候比 $j$ 优？

$T_i L_j\lt T_jL_i$。也就是拿 $i$ 造成的 $j$ 流失少一些。

然后考虑怎么选这些物品。

只能选一次，流失的直接算进价值里就行。也就是一个01背包。

所以按照上面的不等式排序，然后进行01背包就可。

然而，要是考虑到01背包时间维的大小，这题还是过不了。

这么想，要是每个都不流失，那么需要拿 $\displaystyle\sum_{i=1}^n\max\{ T_i\}$ 也就是 $10^6$ 这么长时间。

但是，要是不流失的话，还关心时间干什么。直接放到最后一个个全拿走就好了啊。

所以把那些不流失的先拎出来，答案加上其 $E_i$。

剩下的接着做01背包。

这样就可以优化掉很多空间和时间了。剩下的都是要流失的，流失得最慢的也只需要 $\displaystyle\max\left\{\frac {E_i}{L_i}\right\}=10000$ 就流完了。所以如果流得最慢的这个是最后一个被拿的，那么需要花的时间就是 $\displaystyle\max\left\{\frac {E_i}{L_i}\right\}+\max\{T_i\}=20000$。

啊，所以01背包数组开 $20000$ 就够了。

这就A了。

此题是ysq学长特意出的来教我们卡空间卡时间，学到了学到了。
