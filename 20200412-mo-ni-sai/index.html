题目描述

用$m$种颜色的彩球装点$n$层的圣诞树。圣诞树的第$i$层恰由$l[i]$个彩球串成一行，且同一层内的相邻彩球颜色不同，同时相邻两层所使用彩球的颜色集合不同。求有多少种装点方案，答案对$p$取模。

只要任一位置上的彩球颜色不同，就算作不同的方案。

第一行三个整数$n,m,p$，表示圣诞树的层数、彩球的颜色数和取模的数。

接下来一行包含$n$个整数，表示$l[i]$。

输出格式

一个整数表示答案。

输入输出样例

输入 #1

输出 #1

输入 #2

输出 #2

说明/提示

对于$30\%$的数据，$m \le 2,\ \sum l[i] \le 10$

对于$60\%$的数据，$n \le 20,\ l[i] \le 100$

对于$100\%$的数据，$1 \le n,m \le 10^6,\ 2 \le p \le 10^9,\ 1 \le l[i] \le 5000,\ \sum l[i] <= 10^7$

题解

题目来源：CodeForces 140 E

考察要点：动态规划、组合数学

先考虑小问题：恰用$j$种颜色布置一行$i$个球的方案数$dp[i][j]$。

用类似于最小表示法的思想，我们要求$x$号颜色的首次出现位置必须比$x+1$号颜色的早，这样一来将所求得的方案数乘以颜色的全排列数$j!$就是原来的方案数。

若前$i-1$个球使用了$j-1$种颜色，则第$i$个球必然使用了第$j$种颜色；若前$i-1$个球已使用了$j$种颜色，则第$i$个球使用的颜色必须与第$i-1$个球不同，所以有$(j-1)$种方案。故可由简单的dp递推而得：

$F[i][j] = F[i-1][j-1] + F[i-1][j] * (j-1)$

现在考虑相邻两层之间的限制，用$d[i][j]$表示前$i$层中第$i$层恰用了$j$种颜色的方案数。因为使用的颜色数不可能多于球数，又总球数不超过$10^7$,故$d[i][j]$的状态总数也不超过$10^7$。

T129180 Colorado Potato Beetle

题目背景

Time Limit: $1s$

Memory Limit: $256MB$

大样例：链接：https://pan.baidu.com/s/1iEJS-1cKyXM3JfDmxdB9fg

题目描述

Old MacDonald有一个农场和大小为$(10^{10} + 1) \times (10^{10} + 1)$平方米的巨大的土豆种植地。这块田地被分成正方形的小块，每块占地一平方米。

Old MacDonald知道科罗拉多土豆甲虫将要入侵，并且会破坏收成。Old MacDonald想在一些土地上喷上杀虫剂。

所以，Old MacDonald去了田里，站在田地中间的那个小块中间并将这个格子撒上杀虫剂。现在，他将要通过一系列的移动来在更多田地喷洒杀虫剂。每次移动的时候，Old MacDonald会在上下左右四个方向中选择一个并移动整数米。当他移动的时候，他会在走过的每个格子上喷洒杀虫剂。也就是说，和Old MacDonald的移动轨迹有任何交点的小块都会被撒上杀虫剂。

当Old MacDonald停止喷洒杀虫剂后，他将他所有的移动记录在一张纸上。现在，他想知道，有多少小块不会受到科罗拉多甲虫入侵的影响。

我们知道，科罗拉多甲虫的入侵按照以下顺序展开。一开始，一些土地边缘的小块被入侵。接着，任何没有撒过杀虫剂且有一个相邻（与其有公共边的）小块被感染的尚未被感染的土地也会被感染。帮助Old MacDonald计算有多少小块不会受到科罗拉多甲虫的感染。

输入格式

第一行包含一个正整数$n$，表示Old MacDonald的移动步数。 接下来$n$行包含了对Old MacDonald移动的描述。其中的第$i$行描述第$i$次移动。每次移动通过“$d_i\ x_i$”的形式给出。$d_i$是一个字母描述移动的方向，（”$L$”,”$R$”,”$U$”,”$D$”分别表示左右上下），$x_i$表示这次移动的距离。

输出格式

一行，不会被科罗拉多甲虫感染的土地数量。

输入输出样例

输入 #1

输出 #1

输入 #2

输出 #2

输入 #3

输出 #3

说明/提示

$20\%$的数据满足$n,x\le 30$

另有$20\%$的数据满足$n=4$

$1\le n\le 1000,\ 1\le x\le 1000000$

题解

离散化后直接bfs…主要难度在实现细节，不多说了。

考虑状态的表示。按原图点数过多，故离散化。
离散化后只有$O(N^2)$个结点，之后直接遍历即可，dfs或bfs。于是复杂度是$O(N^2)$

具体离散化方案：
每个关键字单独作为$1$个格子，两个关键字之间再新增一个格子。

  • 40pts：在上一个的基础上疯狂特判。

T129181 Distinct Paths

题目背景

Distinct Paths

Time Limit: $1s$

Memory Limit: $256MB$

题目描述

小美有一个$n*m$的木板，一些块已经被涂上给出的$k$种颜色中的一种。你需要把每个没涂色的块涂色使得从左上角到右下角的每条路径都不会经过两个颜色一样的块。路径只能向右或向下走。

输出答案%$1000000007$。

输入格式

第一行$3$个整数$n,m,k$。

接下来$n$行，每行$m$个整数表示木板，其中第一行表示最上的$m$块，每行第一个数表示最左列的块。

对于每个整数，如果为$0$，表示未涂色，否则表示涂上颜色的编号。

输出格式

一个答案表示方案数。

输入输出样例

输入 #1

输出 #1

说明/提示

$10\%$的数据满足$n=1$

$30\%$的数据满足$n\le 2$

$1\le n,m\le1000,\ 1\le k\le10$

题解

轮廓线dp。$f[i][j][S]$表示轮廓线到了$i$行$j$列状态为$S$的方案数。S表示每一种颜色最前出现的列的编号，压成一个不超过$6^{10}$的int，用map保存状态。
剪枝就是对于没出现过的颜色在矩形里是等价的，所以可以把这些颜色压在一起，转移时特判即可。
时间复杂度$O(nmkS)$，$S$上限为$6^{10}$，当然远远不到那么多。

直接爆搜。
从上到下，从左到右逐格搜索，并记录从起点出发到轮廓线上每个格子可能会经过的颜色。
中间可能会有许多同构的情况。通过这个剪枝。
先假设地图全为$0$.
此时我们加限制条件：假设已经使用了$tot$种颜色，如果要使用新的颜色，则一定使用第$tot+1$种。
注意此时第$i$种颜色并不再对应原来的第$i$种颜色，而是可以对应于原来任意一种颜色，每一种对应方案都是同构的。
如果地图上已经有染过色的，那么对于这个颜色，我们将其标记为不自由色。
最后每种模式的贡献为：假设用了$tot$种颜色，有$F$种自由色，总共有$K$种颜色，那么答案就是$P(K - (tot - F),F)$，其中$P(m,n)$代表$m$取$n$个的排列方案数。
