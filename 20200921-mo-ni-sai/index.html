题目背景

题目描述

小T要给他的妹妹买礼物。他会不断的买，直到自己身上没有足够的钱来买任何一件礼物为止，他想知道有多少种方案符合他买礼物的方式。

我们认为两种选择方案不同当且仅当它们选取的的物品的集合不同。

因为答案可能很大，你只需要输出答案对$10^7+7$取模的结果即可。

输入输出格式

输入格式

第一行两个正整数 $N,M$，表示可以购买的礼物总数和小T的钱数。 第二行 $N$ 个正整数，表示每件礼物的价格 $C_i$。

输出格式

输出一个数，答案对 $10^7+7$ 取模的结果。

输入输出样例

输入样例 #1

输出样例 #1

输入样例 #2

输出样例 #2

说明

样例解释

样例1共15种方案，把物品按1~6标号，分别是： {1,2,3}, {1,2,4}, {1,2,6}, {1,3,4}, {1,3,6}, {1,4,6}, {1,5}, {2,3,4}, {2,3,6}, {2,4,6}, {2,5}, {3,4,6}, {3,5}, {4,5}, {5,6}。

数据范围

30%的数据，$N=10, M=100$

另外30%的数据，$N=100, M=500$

另外40%的数据，$N=1000，M=1000$

所有的数据，$0 \le C_i \le M$。

数据均为随机生成。

题解

设选了的集合为 $S$，没选的为 $W$，则只要满足 $\displaystyle\sum_{i\in S}C_i\gt M-\min_{j\in W}\{C_j\}$ 就可以统计这个 $S$ 集合进答案了，因为没法再选了。

首先肯定先排个序。

开一个二维 DP 数组 $f_{i,j}$ 表示考虑到了第 $i$ 个，要花 $j$ 这么多钱。

以下是递推做法。

在枚举转移的时候外层是 $1$ 到 $n$，内层是 $0$ 到 $m$（因为是二维的，所以转移顺序就无所谓了。如果开一维数组那就从 $m$ 到 $0$），然后考虑转移。

如果 $i$ 是 $1$ 的话，不管有多少钱，方案数只有一，$f_{i,j}=1$。

以下每句话都建立在之前的情况不成立的情况下，也就是else if。

如果 $j<C_i$，说明现在的钱并不够买当前的物品，所以方案数和 $i-1$ 的是一样的，$f_{i,j}=f_{i-1,j}$。

如果 $j\ge sum_i$，也就是一直到 $i$ 都能买，那么就只能全选上（否则不满足题意），$f_{i,j}=1$。

剩下的情况，就可以考虑买这个物品和不买这个物品，$f_{i,j}=f_{i-1,j}+f_{i-1,j-C_i}$。

递推似乎不太好理解，那可以再递归想一想，考虑 $f_{n,m}$ 这个答案是怎么来的吧。

如果 $m\ge sum_n$，那么只能是 $1$，也就是把所有都选上，因为不可能出现别的情况。

如果不是这样呢？那么可以考虑选前 $n-1$ 的方案数 $f_{n-1,m}$，或者前面某几个不选，把这个选上 $f_{n-1,m-C_n}$。

递归的边界就是 $n=1$，这时候，如果 $m\ge C_1$，就只能把 $1$ 选上，方案数为 $1$，否则连 $1$ 也不能选，方案数也是 $1$。或者 $m<0$，这种直接返回 $0$ 就行，因为不可能有什么方案。

嗯。。。就这样吧。（递归：Recursion、递推：Recurrence）

树(tree)

题目描述

梦游中的你来到了一棵$N$个节点的树上. 你一共做了$Q$个梦, 每个梦需要你从点$u$走到点$v$之后才能苏醒, 由于你正在梦游, 所以每到一个节点后,你会在它连出去的边中等概率地选择一条走过去, 为了确保第二天能够准时到校, 你要求出每个梦期望经过多少条边才能苏醒. 为了避免精度误差, 你要输出答案模$10^9 + 7$的结果。

一共$Q$行, 每行一个整数代表答案

说明

数据范围

对于 40%的数据, $N \le 1000$

对于 100%的数据, $N \le 100000, Q \le 100000$

题解

不知道啥时候加上去的这道题，考前发的 pdf 里没有。

需要维护每个节点到其父节点的期望步数 $f$ 和其父节点到它的期望步数 $g$。

设一个节点的度数为 $d$。

一个节点有 $\displaystyle \frac{1}{d_u}$ 的概率走一步到达 $fa_u$，其期望是 $\displaystyle \frac{1}{d_u}$。

并且对于每个子节点 $son_u$ 都有 $\displaystyle \frac{1}{d_u}$ 的概率到达，花 $1$ 步到 $son_u$，花 $f_{son_u}$ 到 $u$，花 $f_u$ 到 $fa_u$，其期望为 $\displaystyle \frac{1+f_{son_u}+f_u}{d_u}$。

所以得到 $\displaystyle f_u=\frac{1}{d_u}+\frac{1+f_{son_u}+f_u}{d_u}$。

开始颓式子：

发现这就是个类似树形 DP 的玩意。

考虑 $g$ 这个数组咋做。

首先直接走到 $u$：$\displaystyle \frac 1{d_{fa_u}}$。

走到 $fa_u$ 别的 $son_{fa_u}$ 再走回来：$\displaystyle \frac{1+f_{son_{fa_u}}+g_u}{d_{fa_u}}$。

走到 $fa_{fa_u}$ 再走回来：$\displaystyle \frac{1+g_{fa_u}+g_u}{d_{fa_u}}$。

总的：$\displaystyle g_u\cdot d_{fa_u}=1+1+g_{fa_u}+g_u+\sum_{son_{fa_u}}1+g_u+f_{son_{fa_u}} $
