【愚蠢的戴夫】向日葵代沟

题目背景

愚蠢的戴夫和向日葵聊天。

如果向日葵之间产生了代沟，她们就再也不会交流了。

题目描述

一些向日葵站在上排在一列花盆上，她们通过传话的方式互相交流。

话说多了，自然就产生了代沟。一旦有了代沟，两个向日葵就不说话了，话就传不过去。

有$m$个向日葵，$m$个事件。每一个事件都是这样的：

给定一个$k$，排在第$k$的向日葵和排在第$k+1$的向日葵产生了代沟，事件按发生顺序给出。

每一次事件发生后，愚蠢的戴夫想知道有多少个无序数对$(a,b)$，使得排在$a$和排在$b$的向日葵事件发生前可以交流，事件发生后无法交流。

输入输出格式

输入格式

第一行两个整数$n,m$，表示向日葵的个数和事件数量。

接下来$m$行，每行一个正整数，描述一个事件。

输出格式

共$m$行，每一个事件发生后，输出题目所求。

输入输出样例

输入样例 #1

输出样例 #1

输入样例 #2

输出样例 #2

说明

样例$1$解释：

事件一后，$(1,6),(2,6),(3,6),(1,5),(2,5),(3,5),(1,4),(2,4),(3,4)$共九对无序数对满足要求。

事件二后，$(1,3),(2,3)$共两对无序数对满足要求。

事件三后，$(4,6),(4,5)$共两对无序数对满足要求。

  • 对于$10\%$的数据：$n,m\leq 50$。
  • 对于$20\%$的数据：$n,m\leq 200$。
  • 对于$40\%$的数据：$n,m\leq 2000$。
  • 对于$50\%$的数据：$n\leq 10^5, m\leq 2000$。
  • 对于$100\%$的数据：$1\leq m< n\leq 10^5, 1\leq k<n$。

题解

发现每次事件都是在分割整个序列，每次影响到的只有左边最近的分割点 $L$ 到右边最近的分割点 $R$。

那么答案就是 $(k-L+1)\cdot(R-(k+1)+1)$，也就是这两部分任意选。

至于怎么维护这个最近的分割点，暴力肯定不行。所以直接线段树优化到 $\mathcal{O}(\log n)$，查询也就是 $\mathcal{O}(\log n)$，能通过此题。

线段树维护当前节点左边分割点是啥，右边分割点是啥，每次修改都是区间赋值，每次查询都是单点查询。

可以写两套修改查询，也可以用数组的方式直接一套搞定。

我懒。所以写了两套。

【愚蠢的戴夫】树上的僵尸

题目背景

愚蠢的戴夫种了一棵树。

树上有一堆相互讨厌的僵尸。

每一只僵尸都会从树上的任意一个节点走最短路到树上的另外一个节点，并且结束点的编号不小于起始点的编号。

因为僵尸相互讨厌，所以它们在树上的路径不能全部交于一点。

因为僵尸已经很久没有吃到植物了，所以僵尸最多只能经过一定数量的边，不然它们就会饿死在树上。

题目描述

给定$n$个节点的树，以及$m$个僵尸（$m\geq 2$）。

给定$k$，表示僵尸各自最多能走$k$条边（$k\leq n-1$）。

询问所有满足条件的僵尸路径方案数：

  • 结束点编号不小于起始点编号。
  • 不存在所有僵尸的路径交于一点。
  • 僵尸各自的路径包含的边数不超过$k$。

答案对$998244353$取模。

输入输出格式

输入格式

第一行三个整数$n,m,k$，表示树的节点个数，僵尸个数以及僵尸最多能走$k$条边。

接下来$n-1$行，每行两个数$a_i,b_i$，表示编号为$a_i$的节点和编号为$b_i$的节点之间有一条边。

保证形成的是一棵树。

输出格式

一个整数，表示满足条件的方案数。

输入输出样例

输入样例 #1

输出样例 #1

输入样例 #2

输出样例 #2

说明

共$20$个测试点，每个点$5$分

  • 测试点$1-2$： $n\leq 10 , m=2, k\leq 10$ 。
  • 测试点$3-4$： $n\leq 100, m\leq 3, k\leq 50$；保证树的形态为一条链，且链上的编号为$1,2,…,n。$
  • 测试点$5-6$： $n,m,k\leq 10^6$；保证树的形态为一条链，且链上的编号为$1,2,…,n$ 。
  • 测试点$7-8$：$n,m\leq 10^6, k=0$。
  • 测试点$9-10$： $n,m\leq 10^6, k=n-1$。
  • 测试点$11-14$： $n\leq 10^3, m\leq 10^{6}, k\leq 50$。
  • 测试点$15-20$： $n,k\leq 10^3, m\leq 10^{6}$。

【愚蠢的戴夫】豌豆射手

题目背景

愚蠢的戴夫种了一些豌豆射手。

愚蠢的戴夫只会把豌豆射手种在一列草坪上，并且因为戴夫不懂得种植向日葵，所以他并没有足够的阳光使得这一列都放满豌豆射手。

但这些豌豆射手除了会向前喷射豌豆，还会向左右喷射豌豆，并且向左右喷射豌豆有一个飞行距离限制。

一大波僵尸即将来临，戴夫想要知道，有多少种安排豌豆射手的方案，使得没有两个豌豆射手会相互攻击。

题目描述

现在有$n$个豌豆射手，要把它们放在长度为$d$的一列草坪上。

每一个豌豆射手都有一个攻击半径$r_i$，如果将这个豌豆射手放在$pos$位置，那么$[pos-r_i+1,pos+r_i-1]$将不能有其他的豌豆射手。

戴夫要把这$n$个豌豆射手放在长度为$d$的草坪上，使得它们不会相互攻击，求方案数。答案对$10^9+7$取模。

输入输出格式

输入格式

第一行两个整数$n,d$，表示有$n$个豌豆射手，草坪的长度为$d。$

第二行$n$个整数$r_i$，表示每一个豌豆射手的攻击半径。

输出格式

一行一个整数，表示合法的方案数。
