【数据范围】

对于 10% 的数据，n ≤ 20；

对于 25% 的数据，n ≤ 200；

对于 55% 的数据，n − m ≤ 200；

对于 100% 的数据，1 ≤ m ≤ n ≤ 10^5, n − m ≤ 2000。

题解

复杂度肯定和 $n-m$ 挂钩，不然为啥给 $n-m$ 的范围。

先记录一下 $s$ 内部左右扩号消完（匹配就消）剩的扩号（形如 )))((( 因为只要形状不一样就会被消掉）里的左括号和右括号数量，分别记为 $cntL$ 和 $cntR$。

考虑一个 $\mathcal{O}\left((n-m)^2\right)$ 的 DP，$f_{i,j,0}$ 代表前 $i$ 个括号里面左括号减右括号为 $j$ 个的方案数，这里就钦定左括号数量不小于右括号数量（然后后文枚举的时候就枚举 $p$ 的状态，也就是 $s$ 左边的状态，必须要让 $p$ 消完以后的左括号数量不小于 $cntR$ 才能符合条件，不然枚举出负数了谁负责）。

如果求出这个的话，$p$ 的状态确定了，那么对应的 $q$ 的状态也确定了（因为从左边 $i,j$ 就可以推出右边的 $i,j$）。

然后枚举 $p$ 的状态，左右一乘，就行了。

初始化 $f_{0,0,0}=1$。

然后对于 $i\gt 0$ 的 $f_{i,0,0}$，由于钦定了左括号不小于右括号，所以只能让 $i$ 这里是右括号，也就是 $f_{i,0,0}=f_{i-1,1,0}$。

其他的话考虑第 $i$ 个可能是什么括号，如果是右括号那就从 $f_{i-1,j+1,0}$ 转移，如果是左括号那就从 $f_{i-1,j-1,0}$ 转移，$f_{i,j,0}=f_{i-1,j+1,0}+f_{i-1,j-1,0}$。

可能有细心的同学发现了，我 DP 状态是三维，但是全程只用到了两维。

于是同理可证，设 $f_{i,j,1}$ 为前 $i$ 个括号里面右括号减左括号为 $j$ 个的方案数，则有转移：

$\left\{\begin{array}{l}f_{0,0,1}=1，\\ f_{i,0,1}=f_{i-1,1,1},\\ f_{i,j,1}=f_{i-1,j+1,1}+f_{i-1,j-1,1}\end{array}\right.$。

完全一样不是吗？这是为什么呢？

因为 任何一个左括号多于右括号的序列 倒过来就是 右括号多于左括号的序列。于是。。。状态数完全一样。于是就能把第三维去掉直接用两维数组。

然后枚举 $p$ 的状态 $i\in[0,n-m],j\in[0,i]$，也就是 $p$ 最少有 $0$ 个，最多有 $n-m$ 个扩号，这里面左括号减右括号最多是 $i$（也就是全是左括号）。

如果 $p$ 有 $i$ 个扩号，那么 $q$ 就有 $n-m-i$ 个扩号；如果 $p$ 的左括号减右括号数为 $j$，总的是 $0$，还有 $s$ 的左括号减右括号是 $cntL-cntR$，于是 $q$ 的左括号减右括号数就是 $0-j-cntL+cntR$，但是由于 $q$ 这序列要承接 $s$ 多出来的左括号，所以 $0-j-cntL+cntR$ 这个数一定小于等于 $0$，也就是 $j+cntL-cntR$ 就是 $q$ 的右括号减左括号数。于是乎，$p$ 的 $i,j$ 状态所造成的贡献就是 $f_{i,j,0}\times f_{n-m-i,j+cntL-cntR,1}$（为了便于理解还是把第三维加上）。

于是需要注意特判一下不合法的情况，对应 $j+cntL-cntR$ 这个下标要大于等于 $0$，还要保证 $p$ 多出来的左括号能匹配上 $s$ 多出来的右括号，即 $j\ge cntR$，其实这个解决办法就是直接让 $j$ 从 $cntR$ 枚举就行。

简单的期望（exp）

题目描述

给定一个变量$ x $的初始值。

依次执行$ n $次操作，每次操作有$ p \% $的概率令$ x = x × 2，(100 − p)\% $的概率令$ x = x + 1 $。

假设最后得到的值为$ w $，令$ d $为$ w $的质因数分解中$ 2 $的次数，求$ d $的期望。

输入输出格式

输入格式

第一行三个整数$ x, n, p $，含义见题目描述。

输出格式

一行一个实数，表示$ d $的期望。

如果你的答案与标准答案的误差不超过$ 10^{-6} $，则判定为正确。

【数据范围】

对于$20\%$的数据，$n\leqslant 20$；

对于$30\%$的数据，$n\leqslant 50$；

对于$50\%$的数据，$n\leqslant 100$；

对于$100\%$的数据，$x\leqslant 10^9,n\leqslant 200,0\leqslant p\leqslant 100$。

题解

首先，质因数分解中$2$的个数即为二进制表示下末尾$0$的个数。

考虑$DP$，设$dp[i][s][j][0/1]$表示进行第$i$次操作后，当前数的二进制表示后$8$位为$s$，第九位为$0/1$，从第九位开始往前连续$j$位相同的概率。

先来解释后$8$位的原因，因为至少要$2^8+1$次操作才可以向$9$进位，而操作数只有$200$，所以不会对答案造成贡献。

再来解释$j$的用途，假如后$8$位都是$1$第九位也是$1$，那么我们在进行$+1$的操作后会变成$1000……$，而$j$是为了计算出这个$1$的位置。

转移一共有$8$个，不再一一赘述，具体可以看代码。

时间复杂度：$\mathcal{O}(2^8n^2)$。
