P2422 良好的感觉

题目描述

kkk 做了一个人体感觉分析器。每一天，人都有一个感受值 $A_i$，$A_i$ 越大，表示人感觉越舒适。在一段时间 $\left[i, j\right]$ 内，人的舒适程度定义为 $\left[i, j\right]$ 中最不舒服的那一天的感受值 $\times$ $\left[i, j\right]$中每一天感受值的和。现在给出 kkk 在连续 $N$ 天中的感受值，请问，在哪一段时间，kkk 感觉最舒适？

输入输出格式

输入格式

第一行为 $N$，代表数据记录的天数。

第二行 $N$ 个整数，代表每一天的感受值。

输出格式

一行，表示在最舒适的一段时间中的感受值。

输入输出样例

输入样例 #1

输出样例 #1

说明

kkk 最开心的一段时间是第 $3$ 天到第 $5$ 天，开心值：$(6+4+5)\times4=60$。

对于 $30\%$ 的数据，$1\le N\le 100$。

对于 $70\%$ 的数据，$1\le N\le 2000$。

对于 $100\%$ 的数据，$1\le N\le 100000$，$1\le \texttt{感受值}\le 1000000$。

题解

可以发现这个式子要求的就是一个 $(sum_i-sum_{j-1})\times \min_{j,i}$，这种东西怎么维护呀？

可以发现一个数作为最小值的话，那么它能对答案贡献的最大值是 $a_x\times (sum_{i}-sum_{j-1}),x\in[i,j],\forall k\in[i,j]\ a_k\ge a_x$。

说人话就是包含 $a_x$ 的最长区间满足每个数都不比 $x$ 小。

于是这启发我们可以找到 $x$ 左边比它小的那个和右边比它小的那个。

那么就可以考虑一个单调队列了。因为其并没有弹队头的操作，所以我们可以称其为单调栈。

维护一个单调不降的单调队列，当队尾出队时，当前元素即为队尾元素右边第一个比它小的。

当一个元素被插入队尾时，其前驱元素即为左边第一个比它小的。

于是做完了。

至于没有左边右边比它小的元素的处理，放进去 $a_0=a_{n+1}=0$ 就行。

其实前文叙述有个问题，在出队时候没有考虑到相等元素，可能会出现一大堆相等元素堆在队尾的情况。这种情况对应一整个区间等于最小值的有好几个。那么这样在记录元素的前驱时就会出问题。

不过由于这道题的特殊性，答案并不会变化。

因为即使第一个最小值后面的数的答案都是错的，第一个最小值处的答案也是对的，它的前驱和后继都是对的。

于是又做完了。
