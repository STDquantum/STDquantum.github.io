又名可持久化线段树

<!--more-->

以下定义引自wikipedia

> **可持久化线段树**（在中国国内信息学竞赛社区中又称**总书记树**、**主席树**或**函数式线段树**）是一种**可持久化数据结构**（英语：Persistent data structure）。由于引入者黄嘉泰姓名的缩写与前中共中央总书记、国家主席胡锦涛(H.J.T.)相同，因此这种数据结构也可被称为**总书记树**或**主席树**。这种数据结构在普通线段树的基础之上支持查询某个历史版本，同时时间复杂度与线段树是同级，空间复杂度相较而言更高。
>
> 与大部分可持久化数据结构类似，可持久化线段树尽可能多地共用先前某一个版本的结点，从而节省大量的空间与时间。

# 板子

也不能算板子吧？

- 数组

  ```cpp
  int top, cnt, root[N];
  /* 
  top: 想要建的树的值域
  cnt: 维护节点编号
  root: 每个根的编号
  */
  struct T {
    int val, ls, rs;
    /*
    val: 要维护的值
    ls,rs: 左右儿子
    */
  } t[N << 4];
  ```

- 如果一开始就有值或需要一种全空的情况，需要建一棵初始化的树。

  调用：

  ```cpp
  build(root[0]);
  ```

  代码：

  ```cpp
  void build(int &New, int l = 1, int r = top) {
    New = ++cnt;
    if (l == r) return; // 这个地方可以维护叶子节点的值
    build(t[New].ls, l, mid);
    build(t[New].rs, mid + 1, r);
  }
  ```

- 需要在第 $las$ 个版本上做一些修改变成 $cur$

  调用：

  ```cpp
  modify(root[cur], root[las], val);
  // val可以是想改的任意东西，但一般都是个position
  ```

  代码：

  ```cpp
  void modify(int &New, int Now, int pos, int l = 1, int r = top) {
    New = ++cnt;
    if (l == r) return; // 可以在这附近维护一些东西
    if (mid >= pos) { // 需要修改左子树内的一个链
      modify(t[New].ls, t[Now].ls, pos, l, mid);
      t[New].rs = t[Now].rs;
    } else {
      modify(t[New].rs, t[Now].rs, pos, mid + 1, r);
      t[New].ls = t[Now].ls;
    }
  }
  ```

- 需要查询一个什么值：维护的数组的值或者是一个什么前缀和啥的

  调用：

  ```cpp
  query(root[cur], val);
  ```

  代码：

  ```cpp
  int query(int Now, int pos, int l = 1, int r = n) {
    if (l == r) return t[Now].val;
    if (mid >= pos) {
      return query(t[Now].ls, pos, l, mid);
    } else {
      return query(t[Now].rs, pos, mid + 1, r);
    }
  }
  ```

# 例题

## [P3834【模板】可持久化线段树 2（主席树）](https://www.luogu.com.cn/problem/P3834)

### 题目背景

这是个非常经典的主席树入门题——静态区间第 $k$ 小。 

**数据已经过加强，请使用主席树。同时请注意常数优化**。

### 题目描述

如题，给定 $n$ 个整数构成的序列 $a$，将对于指定的闭区间 $[l, r]$ 查询其区间内的第 $k$ 小值。

### 输入输出格式

#### 输入格式

第一行包含两个整数，分别表示序列的长度 $n$ 和查询的个数 $m$。 

第二行包含 $n$ 个整数，第 $i$ 个整数表示序列的第 $i$ 个元素 $a_i$。 

接下来 $m$ 行每行包含三个整数 $ l, r, k$ ， 表示查询区间 $[l, r]$ 内的第 $k$ 小值。

#### 输出格式

对于每次询问，输出一行一个整数表示答案。

### 输入输出样例

#### 输入样例 #1

```
5 5
25957 6405 15770 26287 26465 
2 2 1
3 4 1
4 5 1
1 2 2
4 4 1
```

#### 输出样例 #1

```
6405
15770
26287
25957
26287
```

### 说明

#### 样例 1 解释

 $n=5$，数列长度为 $5$，数列从第一项开始依次为$\{25957, 6405, 15770, 26287, 26465\}$。

- 第一次查询为 $[2, 2]$ 区间内的第一小值，即为 $6405$。
- 第二次查询为 $[3, 4]$ 区间内的第一小值，即为 $15770$。
- 第三次查询为 $[4, 5]$ 区间内的第一小值，即为 $26287$。
- 第四次查询为 $[1, 2]$ 区间内的第二小值，即为 $25957$。
- 第五次查询为 $[4, 4]$ 区间内的第一小值，即为 $26287$。

#### 数据规模与约定 

- 对于 $20\%$ 的数据，满足 $1 \leq n,m \leq 10$。
- 对于 $50\%$ 的数据，满足 $1 \leq n,m \leq 10^3$。
- 对于 $80\%$ 的数据，满足 $1 \leq n,m \leq 10^5$。
- 对于 $100\%$ 的数据，满足 $1 \leq n,m \leq 2\times 10^5$，$|a_i| \leq 10^9$，$1 \leq l \leq r \leq n$，$1 \leq k \leq r - l + 1$。

```cpp
using namespace std;

namespace STDquantum {

template <typename T> inline void read(T &x) {
  char ch;
  int f = 1;
  while (ch = getchar(), !isdigit(ch))
    if (ch == '-') f = -1;
  x = (ch ^ 48);
  while (ch = getchar(), isdigit(ch)) x = x * 10 + (ch ^ 48);
  x *= f;
}
template <typename T> inline void write(T x) {
  static int stk[100], top = 0;
  if (x == 0) return (void)putchar('0');
  if (x < 0) x = -x, putchar('-');
  while (x) stk[++top] = x % 10, x /= 10;
  while (top) putchar(stk[top--] + '0');
}

const int N = 2e5 + 10;

#define mid ((l + r) >> 1)
int top, cnt, root[N];
struct T {
  int val, ls, rs;
} t[N << 5];
void build(int &New, int l = 1, int r = top) {
  New = ++cnt;
  if (l == r) return;
  build(t[New].ls, l, mid), build(t[New].rs, mid + 1, r);
}
void modify(int &New, int Now, int pos, int l = 1, int r = top) {
  New = ++cnt;
  t[New].val = t[Now].val + 1;
  if (l == r) return;
  if (mid >= pos) {
    modify(t[New].ls, t[Now].ls, pos, l, mid);
    t[New].rs = t[Now].rs;
  } else {
    modify(t[New].rs, t[Now].rs, pos, mid + 1, r);
    t[New].ls = t[Now].ls;
  }
}
int query(int New, int Now, int k, int l = 1, int r = top) {
  if (l == r) return l;
  int tmp = t[t[New].ls].val - t[t[Now].ls].val;
  if (k <= tmp) {
    return query(t[New].ls, t[Now].ls, k, l, mid);
  } else {
    return query(t[New].rs, t[Now].rs, k - tmp, mid + 1, r);
  }
}
int n, m, a[N], b[N];
inline void main() {
  read(n), read(m);
  for (int i = 1; i <= n; ++i) { read(a[i]), b[i] = a[i]; }
  sort(b + 1, b + n + 1);
  top = unique(b + 1, b + n + 1) - b - 1;
  build(root[0]);
  for (int i = 1, pos; i <= n; ++i) {
    pos = lower_bound(b + 1, b + top + 1, a[i]) - b;
    modify(root[i], root[i - 1], pos);
  }
  for (int i = 1, l, r, k, pos; i <= m; ++i) {
    read(l), read(r), read(k);
    pos = query(root[r], root[l - 1], k);
    write(b[pos]), putchar('\n');
  }
}
```

## Count on a tree

[P2633 Count on a tree](https://www.luogu.com.cn/problem/P2633)

### 题目描述

给定一棵 $n$ 个节点的树，每个点有一个权值。有 $m$ 个询问，每次给你 $u,v,k$，你需要回答 $u \text{ xor last}$ 和 $v$ 这两个节点间第 $k$ 小的点权。

其中 $\text{last}$ 是上一个询问的答案，定义其初始为 $0$，即第一个询问的 $u$ 是明文。

### 输入输出格式

#### 输入格式

第一行两个整数 $n,m$。

第二行有 $n$ 个整数，其中第 $i$ 个整数表示点 $i$ 的权值。

后面 $n-1$ 行每行两个整数 $x,y$，表示点 $x$ 到点 $y$ 有一条边。

最后 $m$ 行每行两个整数 $u,v,k$，表示一组询问。

#### 输出格式

$m$ 行，每行一个正整数表示每个询问的答案。

### 输入输出样例

#### 输入样例 #1

```
8 5
105 2 9 3 8 5 7 7
1 2
1 3
1 4
3 5
3 6
3 7
4 8
2 5 1
0 5 2
10 5 3
11 5 4
110 8 2
```

#### 输出样例 #1

```
2
8
9
105
7
```

### 说明

【数据范围】 

对于 $100\%$ 的数据，$1\le n,m \le 10^5$。

暴力自重。。。 

来源：bzoj2588 Spoj10628. 

本题数据为洛谷自造数据，使用[CYaRon](https://github.com/luogu-dev/cyaron)耗时5分钟完成数据制作。

### 题解

来看一般的主席树（指洛谷板子）都干了点啥，求一个区间内数的个数，这样就可以找到第 $k$ 大（或者小）的数。

而如果搬到树上，是不是就可以维护一些诸如两个点之间的链上第 $k$ 大的点了？

首先维护每个节点到根的链上每个区间内数的个数，然后查 $u,v,k$ 的时候，查询${\rm query}(root_u, root_v, root_{ {\rm lca}(u, v)}, root_{fa[{\rm lca}(u, v)]}, k)$。

然后在统计 $sum$ 的时候，统计 $sum_u+sum_v-sum_{ {\rm lca}(u, v)}-sum_{fa[{\rm lca}(u, v)]}$，这是什么原因呢？因为 $sum_u+sum_v$ 包括了两倍的 $sum_{ {\rm lca}(u, v)}$，我们需要保留 ${\rm lca}(u, v)$ 的信息，所以删掉一个 $sum_{ {\rm lca}(u, v)}$，再删掉一个 $sum_{fa[{\rm lca}(u, v)]}$，就得到了我们想要的 $u$ 到 $v$ 路径上的所有信息。

套上面的板子，dfs的时候维护节点到根的链，继承父节点的线段树就行。

All in all，<a href="./#Count-on-a-tree">Count on a tree</a> 是把主席树的模板搬到了树到当前节点的链上。

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

template <typename T> inline void read(T &x) {
  char ch;
  int f = 1;
  while (ch = getchar(), !isdigit(ch)) {
    if (ch == '-') f = -1;
  }
  x = (ch ^ 48);
  while (ch = getchar(), isdigit(ch)) { x = x * 10 + (ch ^ 48); }
  x *= f;
}

const int N = 1e6 + 10;

int tot, head[N];
struct E {
  int next, to;
} e[N];
inline void add(int x, int y) {
  e[++tot] = (E){head[x], y}, head[x] = tot;
  e[++tot] = (E){head[y], x}, head[y] = tot;
}

#define mid ((l + r) >> 1)
int top, cnt, root[N];
struct T {
  int sum, ls, rs;
} t[N << 4];
void build(int &New, int l = 1, int r = top) {
  New = ++cnt;
  if (l == r) return;
  build(t[New].ls, l, mid);
  build(t[New].rs, mid + 1, r);
}
void modify(int &New, int Now, int pos, int l = 1, int r = top) {
  New = ++cnt;
  t[New].sum = t[Now].sum + 1;
  if (l == r) return;
  if (mid >= pos) {
    modify(t[New].ls, t[Now].ls, pos, l, mid);
    t[New].rs = t[Now].rs;
  } else {
    modify(t[New].rs, t[Now].rs, pos, mid + 1, r);
    t[New].ls = t[Now].ls;
  }
}
int query(int U, int V, int LCA, int LCAF, int pos, int l = 1,
          int r = top) {
  if (l == r) return l;
  int sum = t[t[U].ls].sum + t[t[V].ls].sum - t[t[LCA].ls].sum -
            t[t[LCAF].ls].sum;
  if (sum >= pos) {
    return query(t[U].ls, t[V].ls, t[LCA].ls, t[LCAF].ls, pos, l,
                 mid);
  } else {
    return query(t[U].rs, t[V].rs, t[LCA].rs, t[LCAF].rs, pos - sum,
                 mid + 1, r);
  }
}

int n, m, a[N], b[N];
int lg[N], dep[N], fa[N][21];
void dfs(int s, int f) {
  dep[s] = dep[f] + 1, fa[s][0] = f;
  modify(root[s], root[f], a[s]);
  for (int i = 1; i <= lg[dep[s]]; ++i)
    fa[s][i] = fa[fa[s][i - 1]][i - 1];
  for (int i = head[s]; i; i = e[i].next) {
    if (e[i].to != f) dfs(e[i].to, s);
  }
}
inline int lca(int x, int y) {
  if (dep[x] < dep[y]) swap(x, y);
  while (dep[x] > dep[y]) x = fa[x][lg[dep[x] - dep[y]] - 1];
  if (x == y) return x;
  for (int k = lg[dep[x]] - 1; ~k; --k) {
    if (fa[x][k] != fa[y][k]) x = fa[x][k], y = fa[y][k];
  }
  return fa[x][0];
}

inline void main() {
  int las = 0;
  read(n), read(m);
  for (int i = 1; i <= n; ++i) read(a[i]), b[i] = a[i];
  sort(b + 1, b + n + 1);
  top = unique(b + 1, b + n + 1) - b - 1;
  for (int i = 1; i <= n; ++i)
    a[i] = lower_bound(b + 1, b + top + 1, a[i]) - b;
  build(root[0]);

  for (int i = 1, x, y; i < n; ++i) read(x), read(y), add(x, y);
  for (int i = 1; i <= n; ++i)
    lg[i] = lg[i - 1] + (1 << lg[i - 1] == i);
  dfs(1, 0);
  for (int i = 1, x, y, z, LCA; i <= m; ++i) {
    read(x), read(y), read(z);
    x ^= las, LCA = lca(x, y);
    las = b[query(root[x], root[y], root[LCA], root[fa[LCA][0]], z)];
    printf("%d\n", las);
  }
}

}  // namespace STDquantum

int main() {
#ifndef ONLINE_JUDGE
#ifdef LOCAL
  freopen("D:\\OI\\STDquantum\\test.in", "r", stdin);
  freopen("D:\\OI\\STDquantum\\test.out", "w", stdout);
#endif
#ifndef LOCAL
  freopen("P2633_Count_on_a_tree.in", "r", stdin);
  freopen("P2633_Count_on_a_tree.out", "w", stdout);
#endif
#endif
  STDquantum::main();
  return 0;
}
```

## [CQOI2015]任务查询系统

[P3168 [CQOI2015]任务查询系统](https://www.luogu.com.cn/problem/P3168)

### 题目描述

最近实验室正在为其管理的超级计算机编制一套任务管理系统，而你被安排完成其中的查询部分。

超级计算机中的任务用三元组 $(s_i,e_i,p_i)$ 描述，$(s_i,e_i,p_i)$ 表示任务从第 $s_i$ 秒开始，在第 $e_i$ 秒后结束（第 $s_i$ 秒和 $e_i$ 秒任务也在运行），其优先级为 $p_i$。同一时间可能有多个任务同时执行，它们的优先级可能相同，也可能不同。

调度系统会经常向查询系统询问，第 $x_i$ 秒正在运行的任务中，优先级最小的 $k_i$ 个任务（即将任务按照优先级从小到大排序后取前 $k_i$ 个）的优先级之和是多少。

特别的，如果 $k_i$ 大于第 $x_i $秒正在运行的任务总数，则直接回答第 $x_i$ 秒正在运行的任务优先级之和。上述所有参数均为整数，时间的范围在 $[1,n]$ 之间。

### 输入输出格式

#### 输入格式

输入文件第一行包含两个空格分开的正整数 $m$ 和 $n$，分别表示任务总数和时间范围。

接下来 $m$ 行，每行包含三个空格分开的正整数 $s_i,e_i,p_i$，( $s_i \le e_i$ )描述一个任务。

接下来 $n$ 行，每行包含四个空格分开的整数 $x_i,a_i,b_i,c_i$，描述一次查询。

**本题强制在线。**

查询的参数 $k_i$ 需要由公式 $k_i = 1 +(a_i \times \text{pre}+b_i) \bmod c_i$ 计算得到。

其中 $\text{pre}$ 表示上一次查询的结果，定义初始 $\text{pre} = 1$ 。

#### 输出格式

输出共 $n$ 行，每行一个整数，表示查询结果。

### 输入输出样例

#### 输入样例 #1

```
4 3
1 2 6
2 3 3
1 3 2
3 3 4
3 1 3 2
1 1 3 4
2 2 4 3
```

#### 输出样例 #1

```
2
8
11
```

### 说明

样例解释

$k_1 = (1\times 1 + 3)\bmod 2 + 1 = 1$

$k_2 = (1\times 2+3)\bmod 4 + 1 = 2$

$k_3 = (2 \times 8+4)\bmod 3+1 = 3$

【数据范围】 

对于 $100\%$ 的数据，$1\le m,n,s_i,e_i,c_i,a_i,b_i \le 100000$，$1\le p_i \le 10^7$，$x_i$ 为 $1$ 到 $n$ 的一个排列。

### 题解

首先观察这题，简化一下题面。

给出 $m$ 个三元组 $(s,e,p)$，询问 $(x,k)$ 为满足 $s \le x \le e$ 的三元组的前 $k$ 大 $p$ 值之和，强制在线。

对比一下与洛谷模板的区别，在加入元素的时候变成了加入到区间每个位置（区间加），查询变成了查询一个特定位置的前 $k$ 大的和（单点查询）。相应原来是单点加、区间查询。

有没有点眼熟？

在当初学树状数组区间加单点查询的时候，用到了一个叫差分的东西。

同样地，我们在主席树上维护的东西也可以变成一个差分，就把区间加的问题转化为了单点加，然后把所有单点加按横坐标排序，遍历到了这个地方，就把这个数（左端点为加，右端点加一处为减）统计进主席树里。

若是遇到一个点有多次修改的情况，每次修改的时候都新建一棵线段树，这个点的根记录为新的线段树的根（因为不需要之前的了）。

然后呢，还有一个问题，就是查前 $k$ 大的和，这个操作可以通过维护一个区间内的 $sum$ 实现。

而主席树怎么查询每个点的情况呢？

因为主席树上一棵线段树实际上维护的是到开始节点的整个区间的前缀和信息，而我们差分以后，这棵线段树就变成了这一个点单独的信息，只需要在一棵树上遍历寻找就好了。

```cpp
using namespace std;

namespace STDquantum {

template <typename T> inline void read(T &x) {
  char ch;
  while (ch = getchar(), !isdigit(ch)) {}
  x = (ch ^ 48);
  while (ch = getchar(), isdigit(ch)) x = x * 10 + (ch ^ 48);
}
template <typename T> inline void write(T x) {
  static int stk[100], top = 0;
  if (x == 0) return (void)putchar('0');
  if (x < 0) x = -x, putchar('-');
  while (x) stk[++top] = x % 10, x /= 10;
  while (top) putchar(stk[top--] + '0');
}

typedef long long ll;
const int N = 2e5 + 10;

#define ls(x) t[x].ls
#define rs(x) t[x].rs
#define mid ((l + r) / 2)
int n, m, b[N];
int top, cnt, root[N];
struct T {
  int ls, rs, num, sum;
} t[N << 5];
inline void pushup(int x) {
  t[x].num = t[ls(x)].num + t[rs(x)].num;
  t[x].sum = t[ls(x)].sum + t[rs(x)].sum;
}
void build(int &New, int l = 1, int r = top) {
  New = ++cnt;
  if (l == r) return;
  build(t[New].ls, l, mid);
  build(t[New].rs, mid + 1, r);
}
void modify(int &New, int Now, int pos, int num, int l = 1,
            int r = top) {
  New = ++cnt;
  if (l == r) {
    t[New].num = t[Now].num + num;
    t[New].sum = t[Now].sum + num * b[pos]; // sum更新时要加上新扔进来的数字的贡献
    return;
  }
  if (mid >= pos) {
    modify(ls(New), ls(Now), pos, num, l, mid);
    rs(New) = rs(Now);
  } else {
    modify(rs(New), rs(Now), pos, num, mid + 1, r);
    ls(New) = ls(Now);
  }
  pushup(New); // 记得更新大区间的信息
}
ll query(int New, int k, int l = 1, int r = top) {
  if (l == r) return (ll)t[New].sum / t[New].num * k;
    /* 这地方需要说一下：因为长度为1的区间维护的sum就相当于这个数字乘个数，所以就这么求前k个的和 */
  ll tmp = t[ls(New)].num;
  if (k <= tmp) { // 判断左区间够不够前k个
    return query(ls(New), k, l, mid);
  } else {
    return t[ls(New)].sum + query(rs(New), k - tmp, mid + 1, r); // 记得加上左区间的和
  }
}

int tot;
struct E {
  int x, p, sgn;
  E(int X = 0, int P = 0, int Sgn = 0) { x = X, p = P, sgn = Sgn; }
  inline friend bool operator<(const E &a, const E &b) {
    return a.x < b.x;
  }
} e[N];

ll ans = 1;
inline void main() {
  read(n), read(m);
  for (int i = 1, l, r, p; i <= n; ++i) {
    read(l), read(r), read(p);
    e[++tot] = E(l, p, 1), e[++tot] = E(r + 1, p, -1); // 分开考虑左右端点
    b[i] = p; // 离散化用
  }
  sort(b + 1, b + n + 1);
  top = unique(b + 1, b + n + 1) - b - 1;
  build(root[0]);
  sort(e + 1, e + 2 * n + 1);
  for (int i = 1, j = 1, pos; i <= 2 * n; ++i) {
    while (j < e[i].x) { root[j + 1] = root[j], ++j; } // 还没轮到后面修改时直接复制前面的树就行
    if (j == n + 1) break;
    pos = lower_bound(b + 1, b + top + 1, e[i].p) - b;
    modify(root[j], root[j], pos, e[i].sgn);
  }
  for (int i = 1, x, a, b, c, k; i <= m; ++i) {
    read(x), read(a), read(b), read(c);
    k = 1 + (ans * a + b) % c;
    if (k >= t[root[x]].num) { // 特判（如题）
      ans = t[root[x]].sum;
    } else {
      ans = query(root[x], k); // 直接查x这棵树就行
    }
    write(ans), putchar('\n');
  }
}

}
```

## 【模板】二逼平衡树（树套树）

### 题目描述

您需要写一种数据结构（可参考题目标题），来维护一个有序数列，其中需要提供以下操作：

1. 查询k在区间内的排名
2. 查询区间内排名为k的值
3. 修改某一位值上的数值
4. 查询k在区间内的前驱(**前驱定义为严格小于x，且最大的数，若不存在输出-2147483647**)
5. 查询k在区间内的后继(**后继定义为严格大于x，且最小的数，若不存在输出2147483647**) 

\#注意上面两条要求和tyvj或者bzoj不一样，请注意

### 输入输出格式

#### 输入格式

若opt=1 则为操作1，之后有三个数l,r,k 表示查询k在区间[l,r]的排名

若opt=2 则为操作2，之后有三个数l,r,k 表示查询区间[l,r]内排名为k的数

若opt=3 则为操作3，之后有两个数pos,k 表示将pos位置的数修改为k

若opt=4 则为操作4，之后有三个数l,r,k 表示查询区间[l,r]内k的前驱

若opt=5 则为操作5，之后有三个数l,r,k 表示查询区间[l,r]内k的后继

#### 输出格式

对于操作1,2,4,5各输出一行，表示查询结果

#### 输入样例 #1

```
4 2 2 1 9 4 0 1 1
2 1 4 3
3 4 10
2 1 4 3
1 2 5 9
4 3 9 5
5 2 8 5
```

### 输出样例 #1

### 说明

时空限制：2s，128M

$ n,m \leq 5\cdot {10}^4 $保证有序序列所有值在任何时刻满足 $ [0, {10} ^8] $

题目来源：bzoj3196 / Tyvj1730 二逼平衡树，在此鸣谢

此数据为洛谷原创。

（**特别提醒：此数据不保证操作4、5一定存在，故请务必考虑不存在的情况**）

### 题解

先考虑朴素的想法，建立一棵主席树，只有 $1,2,4,5$ 操作的话是 $\mathcal{O}(\log n)$ 的，但是多了一个 $3$，直接暴力把后面的全修改会变成 $\mathcal{O}(n\log n)$。

所以想一下怎么均摊这两个复杂度呢？

我们要维护的是个区间的信息，可以用树状数组套主席树。

类比一下树状数组的做法（参考了[这篇博客](https://blog.csdn.net/star_city_7/article/details/81322006?biz_id=102&utm_term=主席树套树状数组&utm_medium=distribute.pc_search_result.none-task-blog-2~all~sobaiduweb~default-0-81322006&spm=1018.2118.3001.4187)）。

| :----------: | :----------------------------------------------------------: | :----------------------------------------------------------: |
|   题目类型   |                      单点修改，区间求和                      |                  单点修改，区间求第 $k$ 大                   |

|  复杂度想法  | 让 $\mathcal{O}(1)$ 和 $\mathcal{O}(n)$ 都变成 $\mathcal{O}(\log n)$ | 让 $\mathcal{O}(\log n)$ 和 $\mathcal{O}(n\log n)$ 都变成 $\mathcal{O}(\log^2n)$ |
|   解决办法   |    建立树状数组，其中每个点<br/>都是若干个原序列数字的和     | 建立树状数组，树状数组中每棵<br/>权值线段树都记录了若干原序列数字 |
| 节点维护信息 |                 $1:1\\2:1,2\\3:3\\4:1,2,3,4$                 | $1$：只含有第一个数字的权值线段树<br/>$2$：只含有前两个数字的权值线段树<br/>$3$：只含有第三个数字的权值线段树<br/>$4$：含有前四个数字的权值线段树 |

如表格中所说，建立树状数组，树状数组中每棵权值线段树都记录了若干原序列数字。

那么查询某个区间的时候（即 $2$ 操作），只需要取出一些节点，用这些节点总共合成的那个权值线段树查找第 $k$ 小就好了。而实现这个合成的话，需要两个辅助数组 $X,Y$ 来记录到底是树状数组上哪些节点组成了这个大树。至于为什么要两个数组，则是因为查询树状数组上的一段区间需要右端点和左端点减一这两个节点的信息，分别对应一堆线段树，用大的减去小的得到的就是这个区间确切的信息。

$1$ 操作：如果 $k$ 在右区间，加上左区间数的个数，否则往左跑，$l=r$ 答案就加上 $1$。这样最后求出来的就是比 $k$ 小的数个数加一。就是 $k$ 的排名。

$3$ 操作：在树状数组里找需要改的线段树，在原值对应地方减一，并在新值对应地方加一。

$4$ 操作：先找到 $k$ 的排名，若为 $1$，说明没前驱，输出 $-2147483647$，否则查 $rank_k-1$ 的数是啥。

$5$ 操作：先找到 $k+1$ 的排名，然后查一下 $rank_{k+1}$ 的数，如果不是 $x$，就说明 $x+1$ 这个数（也就是 $x$ 离散化后的后继）是存在的，输出这个数即可。还要注意如果 $rank_{k+1}$ 大于了区间长度，就说明 $x+1$ 在序列中根本不存在，自然 $x$ 也就没有后继了。

离散化的时候，要把 $1,3,4,5$ 里的 $k$ 全都塞进离散化数组里，保证查询的正确性。

还有一个小优化，初始序列直接建一棵主席树，修改的时候再用树状数组，会省很多空间，统计贡献的时候都统计上就好了。

```cpp
  int ls, rs, val;
} t[N << 5];

            int r = top) {
  New = ++cnt;
  t[New].val = t[Now].val + w;
  if (l == r) return;
  if (mid >= pos) {
    modify(t[New].ls, t[Now].ls, pos, w, l, mid);
    t[New].rs = t[Now].rs;
  } else {
    modify(t[New].rs, t[Now].rs, pos, w, mid + 1, r);
    t[New].ls = t[Now].ls;
  }
}

inline void Init(int l, int r) { // 查询[l,r]，那么用r的减去l-1的就是想要的部分
  totX = totY = 0;
  int i;
  for (i = l - 1; i; i -= i & -i) X[++totX] = S[i]; // 需要减去的部分
  for (i = r; i; i -= i & -i) Y[++totY] = S[i]; // 需要加上的部分
}
inline void addV(int x, int w) { // 3操作的修改一个数，直接在树状数组上修改即可
  int pos = lower_bound(b + 1, b + top + 1, a[x]) - b;
  for (; x <= n; x += x & -x) modify(S[x], S[x], pos, w);
}

  int s = 0, i;
  for (i = 1; i <= totX; ++i) s -= t[t[X[i]].ls].val; // 减掉的部分
  for (i = 1; i <= totY; ++i) s += t[t[Y[i]].ls].val; // 加上的部分
  s += t[t[New].ls].val - t[t[Now].ls].val; // 记得加上原树的贡献
  return s;
}
inline int rank(int New, int Now, int k, int l = 1, int r = top) { // 查询排名
  if (l == r) return 1;
  int i;
  if (mid >= k) { // 如果mid>=k就说明k这个数在左区间里
    for (i = 1; i <= totX; ++i) X[i] = t[X[i]].ls;
    for (i = 1; i <= totY; ++i) Y[i] = t[Y[i]].ls;
    return rank(t[New].ls, t[Now].ls, k, l, mid);
  } else {
    int s = getSum(New, Now); // 加上左区间的数
    for (i = 1; i <= totX; ++i) X[i] = t[X[i]].rs; 
    for (i = 1; i <= totY; ++i) Y[i] = t[Y[i]].rs;
    return s + rank(t[New].rs, t[Now].rs, k, mid + 1, r);
  }
}
inline int kth(int New, int Now, int k, int l = 1, int r = top) { // 查询第k个（最多返回序列最后一个）
  if (l == r) return b[l];
  int s = getSum(New, Now), i;
  if (k <= s) { // 如果左区间的总数比k大，说明k在左区间
    for (i = 1; i <= totX; ++i) X[i] = t[X[i]].ls;
    for (i = 1; i <= totY; ++i) Y[i] = t[Y[i]].ls;
    return kth(t[New].ls, t[Now].ls, k, l, mid);
  } else {
    for (i = 1; i <= totX; ++i) X[i] = t[X[i]].rs;
    for (i = 1; i <= totY; ++i) Y[i] = t[Y[i]].rs;
    return kth(t[New].rs, t[Now].rs, k - s, mid + 1, r);
  }
}

  for (i = 1; i <= m; ++i) {

    if (op == 1) {

      b[++num] = q[i].z;

    } else if (op == 3) {
      read(q[i].x), read(q[i].y);
      b[++num] = q[i].y;
    } else if (op == 4) {
      read(q[i].x), read(q[i].y), read(q[i].z);
      b[++num] = q[i].z;

      read(q[i].x), read(q[i].y), read(q[i].z);
      b[++num] = q[i].z;

    modify(root[i], root[i - 1], pos, 1);

    if (q[i].op != 3 && q[i].op != 2)
```

## [P3157 [CQOI2011]动态逆序对](https://www.luogu.com.cn/problem/P3157)

### 题目描述

对于序列 $a$，它的逆序对数定义为集合
$$
\{(i,j)\vert i<j \wedge a_i > a_j \}
$$
中的元素个数。

现在给出 $1\sim n$ 的一个排列，按照某种顺序依次删除 $m$ 个元素，你的任务是在每次删除一个元素**之前**统计整个序列的逆序对数。

### 输入输出格式

#### 输入格式

第一行包含两个整数 $n$ 和 $m$，即初始元素的个数和删除的元素个数。

以下 $n$ 行，每行包含一个 $1 \sim n$ 之间的正整数，即初始排列。 

接下来 $m$ 行，每行一个正整数，依次为每次删除的元素。

#### 输出格式

输出包含 $m$ 行，依次为删除每个元素之前，逆序对的个数。

### 输入输出样例

#### 输入样例 #1

```
5 4
1
5
3
4
2
5
1
4
2
```

#### 输出样例 #1

```
5
2
2
1
```

### 说明

【数据范围】 

对于 $100\%$ 的数据，$1\le n \le 10^5$，$1\le m \le 50000$。

【样例解释】 删除每个元素之前的序列依次为：

$$
1,5,3,4,2\\1,3,4,2\\3,4,2\\3,2

普通的树状数组求逆序对大家肯定会吧。

那么删除第一个前的逆序对自然能求出来，此时我们考虑删除第一个元素，减少的逆序对个数就是在它前面的比他大的和在它后面比它小的，于是我们开两个数组来存:

as1[i]表示的就是在i前面而且大于i位置上的元素的元素的个数

as2[i]表示的就是在i后面而且小于i位置上的元素的元素的个数

as1[i]就可以在读入的时候直接处理一下

BIT就是简单的前缀和树状数组.

但是又有一个问题了，我已经删了的还是被计算在内了，怎么办?

我减去它与已经删了的构成的逆序对的个数不就好了吗?

那么我们就做成了什么?

每一次在已经删除的元素里面找[l,r]里面小于某个数的元素的个数，很容易想到主席树来做.

既然是维护前缀和的主席树，那是不是我每修改一个都要向后一直修改?

显然复杂度爆表.

前缀和我们还有一种求法，那就是用树状数组，那我能不能用树状数组来代替这种一般前缀和?

显然是可以的，我们每次修改的时候，按照树状数组那样，一直向上修改，求前缀和也像树状数组那样不断加和就好了.

至此，一个树状数组套主席树解决动态逆序对就完结了.

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

#define lowbit(x) (x & (-x))
const int N = 1e5 + 10;
int n, q, pos[N], a[N];
long long ans, ansl[N], ansr[N];
  /*
  pos: 每个数的位置
  ansl/ansr: 每个数左边有多少比它大的，右边有多少比它小的，
      也就是初始时刻左右两侧对答案的贡献。
  ans: 询问的答案
  */
    
struct T {
  long long t[N]; // 树状数组
  void init() { // 初始化清空
    for (int i = 1; i <= n; ++i) { t[i] = 0; }
  }
  long long query(int x) { // 查询前缀和
    long long ans = 0;
    while (x) {
      ans += t[x];
      x -= lowbit(x);
    }
    return ans;
  }
  void add(int x, int w) { // 加入一个元素
    while (x <= n) {
      t[x] += w;
      x += lowbit(x);
    }
  }
} T;

struct C {
#define mid (l + r >> 1)
  static const int M = N << 6;
  int T[M], ls[M], rs[M], cnt; 
  /*
  T: 每个树状数组的根的编号
  ls/rs: 当前节点左右儿子
  cnt: 根的数量
  */
  long long sum[M];
  void add(int &rt, int l, int r, int x) {
    /* 加入新的节点 */
    if (!rt) { rt = ++cnt; }
    ++sum[rt];
    if (l == r) return;
    if (x <= mid) {
      add(ls[rt], l, mid, x);
    } else {
      add(rs[rt], mid + 1, r, x);
    }
  }
  long long query(int rt, int l, int r, int L, int R) {
    /* 查询[L,R]区间内的数字数量 */
    if (L <= l && r <= R) return sum[rt];
    if (l > R || r < L) return 0;
    return query(ls[rt], l, mid, L, R) +
           query(rs[rt], mid + 1, r, L, R);
  }
  long long find(int rt1, int rt2, int L, int R) {
    /* 在[L,R]内，用rt2部分的贡献剪去rt1部分的贡献，计算前缀和 */
    if (L > R) return 0;
    long long ans = 0;
    while (rt1) {
      ans -= query(T[rt1], 1, n, L, R);
      rt1 -= lowbit(rt1);
    }
    while (rt2) {
      ans += query(T[rt2], 1, n, L, R);
      rt2 -= lowbit(rt2);
    }
    return ans;
  }
#undef mid

} C;

inline void main() {
  scanf("%d%d", &n, &q);
  T.init();
  for (int i = 1; i <= n; ++i) {
    scanf("%d", &a[i]);
    ansl[i] = T.query(n) - T.query(a[i]); 
    /* 查询[a[i]+1,n]内数的数量，也就是出现得比它早还比它大的数的数量 */
    pos[a[i]] = i;
    ans += ansl[i];
    T.add(a[i], 1);
  }
  T.init(); // 清空树状数组
  for (int i = n; i; --i) {
    ansr[i] = T.query(a[i] - 1);
    /* 查询[1,a[i]+1]内数的数量，也就是出现得比它晚还比它小的数的数量 */
    T.add(a[i], 1);
  }
  for (int i = 1, tmp, cur; i <= q; ++i) {
    printf("%lld\n", ans); // 删除前查询
    scanf("%d", &tmp);
    cur = pos[tmp]; // 找到现在元素在之前序列的位置
    ans -= (ansl[cur] + ansr[cur]); // 删除元素对答案做出的贡献
    ans += C.find(0, cur, tmp + 1, n) + 
           C.find(cur, n, 1, tmp - 1);
    /* 计算多减掉的删除点对[tmp+1,n]区间的贡献和对[1,tmp-1]区间的贡献 */
    while (cur <= n) {
      C.add(C.T[cur], 1, n, tmp);
      /* 在cur点前缀和加入tmp，使得之后加上减多了的贡献时能正确计算 */
      cur += lowbit(cur);
    }
  }
}

}  // namespace STDquantum

int main() {
  STDquantum::main();
  return 0;
}
```
