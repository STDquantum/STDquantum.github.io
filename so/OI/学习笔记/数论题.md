# GCD

[P2568 GCD](https://www.luogu.com.cn/problem/P2568)

## 题目描述

给定正整数 $n$，求 $1\le x,y\le n$ 且 $\gcd(x,y)$ 为素数的数对 $(x,y)$ 有多少对。

## 输入输出格式

### 输入格式

只有一行一个整数，代表$n$。

### 输出格式

一行一个整数表示答案。

## 输入输出样例

### 输入样例 #1

```
4
```

### 输出样例 #1

```
4
```

## 说明

#### 样例输入输出 1 解释

对于样例，满足条件的 $(x,y)$ 为 $(2,2)$，$(2,4)$，$(3,3)$，$(4,2)$。 

---

#### 数据规模与约定 

- 对于 $100\%$ 的数据，保证 $1\le n\le10^7$。 

---

来源：bzoj2818。

## 解题

列出式子：
$$
\sum_{i=1}^n\sum_{j=1}^n[\gcd(i,j)=prime]
$$
先考虑枚举质数：
$$
\sum_{p\in\text{prime}}\sum_{i=1}^n\sum_{j=1}^n[\gcd(i,j)=p]
$$
把 $\gcd$ 变成 $1$，同时改变枚举上界。
$$
\sum_{p\in\text{prime}}\sum_{i=1}^{\left\lfloor\frac{n}{p}\right\rfloor}\sum_{j=1}^{\left\lfloor\frac{n}{p}\right\rfloor} [\gcd(i,j)=1]
$$
我们发现，后面那个 $\gcd(i,j)=1$ 和 $\varphi$ 有着千丝万缕的联系。

不妨令 $i\ge j$，则可以改变 $j$ 的枚举范围为 $[1,i]$。然而这样丢掉了 $j> i$ 的答案，而之前 $i,j$ 枚举范围均为 $[1,n]$，是一样的，所以 $j\ge i$ 的答案与 $i\ge j$ 的答案完全相同，所以只考虑 $i\ge j$ 就可以了，再将这个答案乘 $2$。

但是可以发现 $i\ge j$ 与 $j\ge i$ 的取值有交集，也就是 $i=j$ 的情况，是重复计算过的。但在本题中，若 $i=j$，则 $\gcd(i,j)=i$，不会计入答案，所以重复的答案只在 $i=j=1$ 的时候出现，本应是 $1$，记录成了 $2$，所以减去 $1$ 即可。变形为
$$
\sum_{p\in\text{prime}}\left(\sum_{i=1}^{\left\lfloor\frac{n}{p}\right\rfloor}\left(2\sum_{j=1}^i [\gcd(i,j)=1]\right)-1\right)
$$
可以发现 $\displaystyle\varphi(i)=\sum\limits_{j=1}^i [\gcd(i,j)=1]$，也就是最里面的括号里的一个东西，所以把它替换掉。
$$
\sum_{p\in\text{prime}}\left(2\sum_{i=1}^{\left\lfloor\frac{n}{p}\right\rfloor}\varphi(i)-1\right)
$$
现在这个式子已经转化为 $\varphi$ 的前缀和形式，$O(n)$ 筛出 $\varphi$ 以后，求出前缀和，再枚举质数计算答案即可。

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

template <typename T> inline void read(T &x) {
  char ch;
  int f = 1;
  while (ch = getchar(), !isdigit(ch)) {
    if (ch == '-') f = -1;
  }
  x = (ch ^ 48);
  while (ch = getchar(), isdigit(ch)) { x = x * 10 + (ch ^ 48); }
  x *= f;
}
const int N = 1e7 + 10;
typedef long long ll;

ll num, prime[N], phi[N];
bool isPrime[N];
inline void Get(ll n) {
  memset(isPrime, 1, sizeof(isPrime));
  isPrime[1] = 0, phi[1] = 1;
  for (int i = 2; i <= n; ++i) {
    if (isPrime[i]) prime[++num] = i, phi[i] = i - 1;
    for (int j = 1; j <= num && i * prime[j] <= n; ++j) {
      isPrime[i * prime[j]] = 0;
      if (i % prime[j] == 0) {
        phi[i * prime[j]] = phi[i] * prime[j];
        break;
      }
      phi[i * prime[j]] = phi[i] * phi[prime[j]];
    }
  }
}

ll n, ans, sum[N];
inline void main() {
  read(n), Get(n);
  for (int i = 1; i <= n; ++i) { sum[i] = sum[i - 1] + phi[i]; }
  for (int i = 1; i <= num; ++i) {
    ans += 2 * sum[n / prime[i]] - 1;
  }
  printf("%lld\n", ans);
}

}  // namespace STDquantum
```

# YY的GCD

[P2257 YY的GCD](https://www.luogu.com.cn/problem/P2257)

## 题目描述

神犇 YY 虐完数论后给傻× kAc 出了一题

给定 $N, M$，求 $1 \leq x \leq N$，$1 \leq y \leq M$ 且 $\gcd(x, y)$ 为质数的 $(x, y)$ 有多少对。

## 输入输出格式

### 输入格式

第一行一个整数 $T$ 表述数据组数。 接下来 $T$ 行，每行两个正整数，$N, M$。

### 输出格式

$T$ 行，每行一个整数表示第 $i$ 组数据的结果。

## 输入输出样例

### 输入样例 #1

```
2
10 10
100 100
```

### 输出样例 #1

```
30
2791
```

## 说明

$T = 10^4$，$N, M \leq 10^7$。

## 解题

列出式子，假定 $n\lt m$
$$
\sum_{i=1}^n\sum_{j=1}^m[\gcd(i,j)=prime]
$$
枚举质数
$$
\sum_{p\in\text{prime}}\sum_{i=1}^n\sum_{j=1}^m[\gcd(i,j)=p]
$$
把 $p$ 除掉
$$
\sum_{p\in\text{prime}}\sum_{i=1}^{\left\lfloor\frac{n}{p}\right\rfloor}\sum_{j=1}^{\left\lfloor\frac{m}{p}\right\rfloor} [\gcd(i,j)=1]
$$

应用莫比乌斯反演
$$
\sum_{p\in\text{prime}}\sum_{i=1}^{\left\lfloor\frac{n}{p}\right\rfloor}\sum_{j=1}^{\left\lfloor\frac{m}{p}\right\rfloor}\sum_{d\mid \lfloor\frac{n}{p}\rfloor}\mu(d)
$$
把 $d$ 提到前面
$$
\sum_{p\in\text{prime}}\sum_{d=1}^{\lfloor\frac{n}{p}\rfloor}\mu(d)
\sum_{i=1}^{\left\lfloor\frac{n}{dp}\right\rfloor}\sum_{j=1}^{\left\lfloor\frac{m}{dp}\right\rfloor} 1=\sum_{p\in\text{prime}}\sum_{d=1}^{\lfloor\frac{n}{p}\rfloor}\mu(d)
\times \left\lfloor\frac{n}{dp}\right\rfloor\times\left\lfloor\frac{m}{dp}\right\rfloor
$$
这样做的话是 $\mathcal{O}(n+Tn\sqrt{n})$ 的，肯定会T。

枚举 $T=dp$
$$
\sum_{T=1}^n
\sum_{p\mid T,p\in{\rm prime}}\mu\left(\left\lfloor\frac{T}{p}\right\rfloor\right)
\times \left\lfloor\frac{n}{T}\right\rfloor\times\left\lfloor\frac{m}{T}\right\rfloor
$$
后面两个就和 $p$ 没什么关系了，提出来
$$
\sum_{T=1}^n
\left\lfloor\frac{n}{T}\right\rfloor\left\lfloor\frac{m}{T}\right\rfloor
\sum_{p\mid T,p\in{\rm prime}}\mu\left(\left\lfloor\frac{T}{p}\right\rfloor\right)=\sum_{T=1}^n
\left\lfloor\frac{n}{T}\right\rfloor\left\lfloor\frac{m}{T}\right\rfloor
\left(\sum_{p\mid T,p\in{\rm prime}}\mu\left(\left\lfloor\frac{T}{p}\right\rfloor\right)\right)
$$
对这堆东西转化一下思路，枚举贡献，设 $\displaystyle f(T)=\sum_{p\mid T,p\in{\rm prime}}\mu\left(\left\lfloor\frac{T}{p}\right\rfloor\right)$，可以发现每个 $p$ 对 $f(p*i)$ 有 $\mu(i)$ 的贡献。所以枚举质数再枚举另一个因子，总次数是 $\displaystyle\sum_{p}\left\lfloor\frac{N}{p}\right\rfloor,N=10^7$，时间复杂度约为 $\mathcal{O}(N\ln N)$，代入 $N$ 大概是 $1.6\times10^8$，数论题一般常数小，一般都没问题。

然后处理出 $f(T)$ 的前缀和，用数论分块就OK，总时间复杂度 $\mathcal{O}(N\ln N+T\sqrt{N})$。

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

template <typename T> inline void read(T &x) {
  char ch;
  int f = 1;
  while (ch = getchar(), !isdigit(ch)) {
    if (ch == '-') f = -1;
  }
  x = (ch ^ 48);
  while (ch = getchar(), isdigit(ch)) { x = x * 10 + (ch ^ 48); }
  x *= f;
}
const int N = 1e7 + 10;
typedef long long ll;
#define intt register int
int num, prime[N], mu[N], f[N];
bool isPrime[N];
inline void Get(int n) {
  memset(isPrime, 1, sizeof(isPrime));
  isPrime[1] = 0, mu[1] = 1;
  for (intt i = 2; i <= n; ++i) {
    if (isPrime[i]) prime[++num] = i, mu[i] = -1;
    for (intt j = 1; j <= num && i * prime[j] <= n; ++j) {
      isPrime[i * prime[j]] = 0;
      if (i % prime[j] == 0) {
        mu[i * prime[j]] = 0;
        break;
      } else {
        mu[i * prime[j]] = -mu[i];
      }
    }
  }
  for (intt i = 1; i <= num; ++i) {
    for (intt j = 1; j * prime[i] <= n; ++j) {
      f[j * prime[i]] += mu[j];
    }
  }
  for (intt i = 1; i <= n; ++i) { f[i] += f[i - 1]; }
}

int t, n, m;
ll ans;
inline void main() {
  read(t), Get(N - 10);
  while (t--) {
    read(n), read(m);
    if (n > m) swap(n, m);
    ans = 0;
    for (intt l = 1, r; l <= n; l = r + 1) {
      r = min(n / (n / l), m / (m / l));
      ans += 1ll * (f[r] - f[l - 1]) * (n / l) * (m / l);
    }
    printf("%lld\n", ans);
  }
}

}  // namespace STDquantum

int main() {
#ifndef ONLINE_JUDGE
#ifdef LOCAL
  freopen("D:\\OI\\STDquantum\\test.in", "r", stdin);
  freopen("D:\\OI\\STDquantum\\test.out", "w", stdout);
#endif
#ifndef LOCAL
  freopen("P2257_YY's_GCD.in", "r", stdin);
  freopen("P2257_YY's_GCD.out", "w", stdout);
#endif
#endif
  STDquantum::main();
  return 0;
}
```

# LCMSUM - LCM Sum

[SP5971 LCMSUM - LCM Sum](https://www.luogu.com.cn/problem/SP5971)

## 题意翻译

- $T$ 次询问，每次询问给定 $n$，求 $\displaystyle\sum_{i=1}^n\text{lcm}(i,n)$
- $1\le T\le 3\times 10^5$，$1\le n\le 10^6$。

## 题目描述

Given n, calculate the sum LCM(1,n) + LCM(2,n) + .. + LCM(n,n), where LCM(i,n) denotes the Least Common Multiple of the integers i and n.

## 输入输出格式

### 输入格式

The first line contains T the number of test cases. Each of the next T lines contain an integer n.

### 输出格式

Output T lines, one for each test case, containing the required sum.

## 输入输出样例

### 输入样例 #1

```
3
1
2
5
```

### 输出样例 #1

```
1
4
55
```

## 解题

真的好累啊。。不想打太多字了。大部分公式和解说来自OI-WIKI，有的公式我改过。
$$
 \sum_{i=1}^n \frac{i\cdot n}{\gcd(i,n)}
$$
将原式复制一份并且颠倒顺序，然后将 $n$ 一项单独提出，可得
$$
 \frac{1}{2}\cdot \left(\sum_{i=1}^{n-1}\frac{i\cdot n}{\gcd(i,n)}+\sum_{i=n-1}^{1}\frac{i\cdot n}{\gcd(i,n)}\right)+n 
$$
根据 $\gcd(i,n)=\gcd(n-i,n)$，可将原式化为
$$
 \frac{1}{2}\cdot \left(\sum_{i=1}^{n-1}\frac{i\cdot n}{\gcd(i,n)}+\sum_{i=n-1}^{1}\frac{i\cdot n}{\gcd(n-i,n)}\right)+n 
$$
两个求和式中分母相同的项可以合并。
$$
 \frac{1}{2}\cdot \sum_{i=1}^{n-1}\frac{n^2}{\gcd(i,n)}+n 
$$
即
$$
 \frac{1}{2}\cdot \sum_{i=1}^{n}\frac{n^2}{\gcd(i,n)}+\frac{n}{2} 
$$
可以将相同的 $\gcd(i,n)$ 合并在一起计算，故只需要统计 $\gcd(i,n)=d$ 的个数。当 $\gcd(i,n)=d$ 时， $\displaystyle\gcd\left(\frac{i}{d},\frac{n}{d}\right)=1$ ，所以 $\gcd(i,n)=d$ 的个数有 $\displaystyle\varphi(\frac{n}{d})$ 个。

故答案为

$$
\frac{1}{2}\cdot\sum_{d\mid n}\varphi(\frac{n}{d})\cdot\frac{n^2}{d}+\frac{n}{2}
$$

变换求和顺序，设 $\displaystyle d'=\frac{n}{d}$ ，合并公因式，式子化为

$$
\frac{1}{2}n\cdot\left(\sum_{d'\mid n}d'\cdot\varphi(d')+1\right)
$$

设 $\displaystyle \text{H}(n)=\sum_{d\mid n} d\cdot\varphi(d)$ ，已知 $\text{H}$ 为积性函数，于是可以 $\mathcal{O}(n)$ 筛出。每次询问 $\mathcal{O}(1)$ 计算 $\displaystyle \frac{n}{2}\cdot({\rm H}(n)+1)$ 即可。

常见积性函数的筛法和证明见<a href="../ji-xing-han-shu-xue-xi-bi-ji" title="积性函数学习笔记">积性函数学习笔记</a>，在TOC（文章悬浮目录）里可以找到。

这里说一下一些特殊条件下的函数取值（$low(i)$ 表示 $p_1^{a_1}$）：

- ${\rm H}(1)=1$，这个是积性函数所要求的。
- ${\rm H}(n),n\in{\rm prime}$，这时就要考虑 $\text{H}(n)$ 的定义 $\displaystyle \text{H}(n)=\sum_{d\mid n} d\cdot\varphi(d)$，$n$ 是质数所以只有 $d=1,n$ 的情况会统计贡献，也就是 ${\rm H}(n)=1\cdot\varphi(1)+n\cdot\varphi(n)=1+n(n-1)$。
- $low(i)=i$，这说明 $i$ 只有一个质因子，而且想要进入 $low(i)$ 和 $i$ 的对比，必须要有 $i\equiv0\pmod{p_j}$，也就是 $n=i\cdot p_j$ 也只有一个质因子，$\displaystyle \text{H}(n)=\sum_{d\mid n} d\cdot\varphi(d)$ 比 $\displaystyle \text{H}(i)=\sum_{d\mid i} d\cdot\varphi(d)$ 多的只有一个 $n\cdot\varphi(n)=i\cdot p_j\cdot low(i)\cdot (p_j-1)$，代码中的`k`是 $i\cdot p_j$，`low[i]`和`i`是一样的。

只放代码：

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

template <typename T> inline void read(T &x) {
  char ch;
  int f = 1;
  while (ch = getchar(), !isdigit(ch)) {
    if (ch == '-') f = -1;
  }
  x = (ch ^ 48);
  while (ch = getchar(), isdigit(ch)) { x = x * 10 + (ch ^ 48); }
  x *= f;
}

typedef long long ll;
const int N = 1e6 + 10;
int num, prime[N];
ll low[N], H[N];
bool isPrime[N];
inline void Get(int n) {
  memset(isPrime, 1, sizeof(isPrime));
  isPrime[1] = 0, low[1] = H[1] = 1;
  for (int i = 2; i <= n; ++i) {
    if (isPrime[i])
      prime[++num] = i, H[i] = (ll)i * (i - 1) + 1, low[i] = i;
    for (int j = 1, k; j <= num && (ll)i * prime[j] <= n; ++j) {
      isPrime[k = i * prime[j]] = 0;
      if (i % prime[j] == 0) {
        low[k] = low[i] * prime[j];
        if (low[i] == i) {
          H[k] = H[i] + (ll)k * i * (prime[j] - 1);
        } else {
          H[k] = H[i / low[i]] * H[prime[j] * low[i]];
        }
        break;
      }
      H[k] = H[i] * H[prime[j]];
      low[k] = prime[j];
    }
  }
}
int t, n;
inline void main() {
  Get(N - 10);
  for (read(t); t; --t) {
    read(n), printf("%lld\n", n * (H[n] + 1) / 2);
  }
}

}  // namespace STDquantum

int main() {
#ifndef ONLINE_JUDGE
#ifdef LOCAL
  freopen("D:\\OI\\STDquantum\\test.in", "r", stdin);
  freopen("D:\\OI\\STDquantum\\test.out", "w", stdout);
#endif
#ifndef LOCAL
  freopen("SP5971_LCMSUM_-_LCM_Sum.in", "r", stdin);
  freopen("SP5971_LCMSUM_-_LCM_Sum.out", "w", stdout);
#endif
#endif
  STDquantum::main();
  return 0;
}
```
