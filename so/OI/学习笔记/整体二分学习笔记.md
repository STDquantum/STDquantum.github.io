一种离线的分治算法

<!--more-->

部分内容引自大佬博客与OI-WIKI，出处均已标明。

> 可以使用整体二分解决的题目需要满足以下性质：
>
> 1. 询问的答案具有可二分性
>
> 2. **修改对判定答案的贡献互相独立** ，修改之间互不影响效果
>
> 3. 修改如果对判定答案有贡献，则贡献为一确定的与判定标准无关的值
>
> 4. 贡献满足交换律，结合律，具有可加性
>
> 5. 题目允许使用离线算法
>
>    ——许昊然《浅谈数据结构题几个非经典解法》

# 大致介绍

以下内容来自OI-wiki

---

记 $[l,r]$ 为答案的值域，$[L,R]$ 为答案的定义域。（也就是说求答案时仅考虑下标在区间 $[L,R]$ 内的操作和询问，这其中询问的答案在 $[l,r]$ 内）

- 我们首先把所有操作 **按时间顺序** 存入数组中，然后开始分治。
- 在每一层分治中，利用数据结构（常见的是树状数组）统计当前查询的答案和 $mid$ 之间的关系。
- 根据查询出来的答案和 $mid$ 间的关系（小于等于 $mid$ 和大于 $mid$ ）将当前处理的操作序列分为 $q1$ 和 $q2$ 两份，并分别递归处理。
- 当 $l=r$ 时，找到答案，记录答案并返回即可。

需要注意的是，在整体二分过程中，若当前处理的值域为 $[l,r]$，则此时最终答案范围不在 $[l,r]$的询问会在其他时候处理。

---

其实就是把所有询问、插入什么的操作全部混在一起，然后一块分治下去。

# 可怜的狗狗

[P1533 可怜的狗狗](https://www.luogu.com.cn/problem/P1533)

## 题目背景

小卡由于公务需要出差，将新家中的狗狗们托付给朋友嘉嘉，但是嘉嘉是一个很懒的人，他才没那么多时间帮小卡喂狗狗。

## 题目描述

小卡家有N只狗，由于品种、年龄不同，每一只狗都有一个不同的漂亮值。漂亮值与漂亮的程度成反比（漂亮值越低越漂亮），吃饭时，狗狗们会按顺序站成一排等着主人给食物。 可是嘉嘉真的很懒，他才不肯喂这么多狗呢，这多浪费时间啊，于是他每次就只给第i只到第j只狗中第k漂亮的狗狗喂食（好狠心的人啊）。而且为了保证某一只狗狗不会被喂太多次，他喂的每个区间（i,j）不互相包含。

## 输入输出格式

### 输入格式

第一行输入两个数n，m，你可以假设n<300001 并且 m<50001；m表示他喂了m次。 第二行n个整数，表示第i只狗的漂亮值为ai。 接下来m行，每行3个整数i,j,k表示这次喂食喂第i到第j只狗中第k漂亮的狗的漂亮值。

### 输出格式

M行，每行一个整数，表示每一次喂的那只狗漂亮值为多少。

## 输入输出样例

### 输入样例 #1

```
7 2
1 5 2 6 3 7 4
1 5 3
2 7 1
```

### 输出样例 #1

```
3
2
```

## 题解

相当于整体二分的裸题了吧，查询静态区间第$k$小。

先放代码（感谢[AlanSP](https://www.luogu.com.cn/user/237308)神仙）

```cpp
#include <bits/stdc++.h>
#define LOCAL
using namespace std;

namespace STDquantum {

template <typename T> inline void read(T &x) {
  char ch;
  while (ch = getchar(), !isdigit(ch)) {}
  x = (ch ^ 48);
  while (ch = getchar(), isdigit(ch)) { x = x * 10 + (ch ^ 48); }
}

#define lowbit(x) ((x) & (-x)) // 比直接写函数要快不少
#define intt register int // 卡常操作
const int N = 6e5 + 10;
int n, m, o, cnt, a[N], b[N], ans[N], t[N]; // cnt为总操作数
    
/* 树状数组部分，要实现查询第k小，就开权值树状数组 */
inline void add(int x, int w) {
  while (x <= n) { t[x] += w, x += lowbit(x); }
}
inline int query(int x) {
  int tmp = 0;
  while (x) { tmp += t[x], x -= lowbit(x); }
  return tmp;
}

struct Q {
  int l, r, k, id, op;
  /* 
  l,r: op=1时，l为要插入的数；op=2时，l、r为查询的区间范围
  k: op=2时，要查询区间第k小
  op: op=1，插入操作；op=2，查询操作
  */
} q[N], q1[N], q2[N];
inline void solve(int l, int r, int L, int R) { // 分治函数
  if (L > R) return; // 防止越界
  if (l == r) { // 如果二分到了一个可能的答案
    for (intt i = L; i <= R; ++i) { // 枚举每个操作
      /* 若为查询操作，说明找到了答案，直接标记答案 */
      if (q[i].op == 2) ans[q[i].id] = l;
    }
    return;
  }
  intt mid = (l + r) >> 1, cnt1 = 0, cnt2 = 0; 
  /* 分别为  mid:区间中点  cnt1,cnt2:分治到左（右）区间的操作个数 */ 
  for (intt i = L; i <= R; ++i) {
    if (q[i].op == 1) { // 若为插入操作
      if (q[i].l <= mid) { // 如果要插入的值在左区间内
        /* 把当前操作分治到左区间内，并计算在当前区间的贡献 */
        q1[++cnt1] = q[i], add(q[i].id, 1);
      } else {
        /* 把当前操作分治到右区间内 */
        q2[++cnt2] = q[i];
      }
    } else {
      /* tmp为左区间内的数的个数 */
      intt tmp = query(q[i].r) - query(q[i].l - 1);
      if (q[i].k <= tmp) { // 如果查询的第k小肯定在左区间
        q1[++cnt1] = q[i]; // 分治到左区间内查询第k小
      } else {
        q[i].k -= tmp, q2[++cnt2] = q[i]; // 要在右区间查第k-tmp小，分治
      }
    }
  }
  for (intt i = 1; i <= cnt1; ++i) {
    if (q1[i].op == 1) add(q1[i].id, -1); // 消除前面插入操作对树状数组的影响
  }
  for (intt i = 1; i <= cnt1; ++i) {
    /* 把[L,L+cnt1-1]定为左区间查询的操作 */
    q[L + i - 1] = q1[i]; 
  }
  for (intt i = 1; i <= cnt2; ++i) {
    /* 把[L+cnt1-1,R]定为右区间查询的操作 */
    q[L + cnt1 + i - 1] = q2[i];
  }
  solve(l, mid, L, L + cnt1 - 1);
  solve(mid + 1, r, L + cnt1, R);
}
inline void main() {
  read(n), read(m);
    
  /* 离散化操作，减小答案值域（二分次数） */
  for (intt i = 1; i <= n; ++i) { read(a[i]), b[++o] = a[i]; }
  sort(b + 1, b + o + 1);
  o = unique(b + 1, b + o + 1) - b - 1;
  for (intt i = 1; i <= n; ++i) {
    a[i] = lower_bound(b + 1, b + o + 1, a[i]) - b;
  }
    
  for (intt i = 1; i <= n; ++i) {
    /* 把离散化后的a数组放入操作序列中，op=1代表是插入操作 */
    q[++cnt] = (Q){a[i], 0, 0, i, 1};
  }
    
  int l, r, k;
  for (intt i = 1, l, r, k; i <= m; ++i) {
    read(l), read(r), read(k);
    q[++cnt] = (Q){l, r, k, i, 2}; // op=2代表查询操作
  }
  
  /* 开始分治，l=1,r=o，即答案的值域；L=1,R=cnt，即对所有操作进行分治 */
  solve(1, o, 1, cnt);
    
  for (intt i = 1; i <= m; ++i) { printf("%d\n", b[ans[i]]); }
}

}  // namespace STDquantum

int main() {
#ifndef ONLINE_JUDGE
#ifdef LOCAL
  freopen("test.in", "r", stdin);
  freopen("test.out", "w", stdout);
#endif
#ifndef LOCAL
  freopen("P1533_Poor_Dog.in", "r", stdin);
  freopen("P1533_Poor_Dog.out", "w", stdout);
#endif
#endif
  STDquantum::main();
  return 0;
}
```

# [国家集训队]矩阵乘法

[P1527 [国家集训队]矩阵乘法](https://www.luogu.com.cn/problem/P1527)

## 题目描述

给你一个 $n \times n$ 的矩阵，不用算矩阵乘法，但是每次询问一个子矩形的第 $k$ 小数。

## 输入输出格式

### 输入格式

第一行有两个整数，分别表示矩阵大小 $n$ 和询问组数 $q$。 第 $2$ 到第 $(n + 1)$ 行，每行 $n$ 个整数，表示这个矩阵。第 $(i + 1)$ 行的第 $j$ 个数表示矩阵第 $i$ 行第 $j$ 列的数 $$a_{i, j}$$。 接下来 $q$ 行，每行五个整数 $$x_1, y_1, x_2, y_2, k$$，表示一组询问，要求找到以 $$(x_1, y_1)$$ 为左上角，$$(x_2, y_2)$$ 为右下角的子矩形中的第 $k$ 小数。

### 输出格式

对于每组询问，输出一行一个整数表示答案。

## 输入输出样例

### 输入样例 #1

```
2 2
2 1
3 4
1 2 1 2 1
1 1 2 2 3
```

### 输出样例 #1

```
1
3
```

## 说明

#### 数据规模与约定 

- 对于 $20\%$ 的数据，保证 $n \leq 100$，$q \leq 10^3$。

- 对于 $40\%$ 的数据，保证 $n \leq 300$，$q \leq 10^4$。

- 对于 $60\%$ 的数据，保证 $n \leq 400$，$q \leq 3 \times 10^4$。

- 对于 $100\%$ 的数据，保证 $1 \leq n \leq 500$，$1 \leq q \leq 6 \times 10^4$，$0 \leq a_{i, j} \leq 10^9$。

## 题解

这题从一维变到了二维，基本操作还是一样的。

依旧可以用狗狗那道题的思路来做，只不过树状数组变成了二维，别的几乎一毛一样。

```cpp
#include <bits/stdc++.h>
#define LOCAL
using namespace std;

namespace STDquantum {

#define lowbit(x) ((x) & (-x))
const int N = 6e5 + 10, M = 1010;
int n, m, cnt, ans[N], t[M][M];
    
inline void add(int x, int y, int w) {
  int ty;
  while (x <= n) {
    ty = y;
    while (ty <= n) { t[x][ty] += w, ty += lowbit(ty); }
    x += lowbit(x);
  }
}
inline int query(int x, int y) {
  int tmp = 0, ty;
  while (x) {
    ty = y;
    while (ty) { tmp += t[x][ty], ty -= lowbit(ty); }
    x -= lowbit(x);
  }
  return tmp;
}
inline int sum(int lx, int ly, int rx, int ry) {
  int tmp = 0;
  tmp += query(rx, ry) + query(lx - 1, ly - 1);
  tmp -= query(rx, ly - 1) + query(lx - 1, ry);
  return tmp;
}

struct Q {
  int lx, ly, rx, ry, k, id, op;
} q[N], q1[N], q2[N];
inline void solve(int l, int r, int L, int R) {
  if (L > R) return;
  if (l == r) {
    for (int i = L; i <= R; ++i) {
      if (q[i].op == 2) ans[q[i].id] = l;
    }
    return;
  }
  int mid = (l + r) >> 1, cnt1 = 0, cnt2 = 0;
  for (int i = L; i <= R; ++i) {
    if (q[i].op == 1) {
      if (q[i].k <= mid) {
        q1[++cnt1] = q[i], add(q[i].lx, q[i].ly, 1);
      } else {
        q2[++cnt2] = q[i];
      }
    } else {
      int tmp = sum(q[i].lx, q[i].ly, q[i].rx, q[i].ry);
      if (q[i].k <= tmp) {
        q1[++cnt1] = q[i];
      } else {
        q[i].k -= tmp, q2[++cnt2] = q[i];
      }
    }
  }
  for (int i = 1; i <= cnt1; ++i) {
    if (q1[i].op == 1) add(q1[i].lx, q1[i].ly, -1);
  }
  for (int i = 1; i <= cnt1; ++i) { q[L + i - 1] = q1[i]; }
  for (int i = 1; i <= cnt2; ++i) {
    q[L + cnt1 - 1 + i] = q2[i];
  }
  solve(l, mid, L, L + cnt1 - 1);
  solve(mid + 1, r, L + cnt1, R);
}

inline void main() {
  scanf("%d%d", &n, &m);
  int maxx = 0, minn = 0x3f3f3f3f;
  for (int i = 1, x; i <= n; ++i) {
    for (int j = 1; j <= n; ++j) {
      scanf("%d", &x);
      minn = min(minn, x), maxx = max(maxx, x); // Kupi神说的那个值域问题
      q[++cnt] = (Q){i, j, 0, 0, x, 0, 1};
    }
  }
  for (int i = 1, lx, ly, rx, ry, k; i <= m; ++i) {
    scanf("%d%d%d%d%d", &lx, &ly, &rx, &ry, &k);
    q[++cnt] = (Q){lx, ly, rx, ry, k, i, 2};
  }
  solve(minn, maxx, 1, cnt);
  for (int i = 1; i <= m; ++i) { printf("%d\n", ans[i]); }
}

}  // namespace STDquantum

int main() {
#ifndef ONLINE_JUDGE
#ifdef LOCAL
  freopen("test.in", "r", stdin);
  freopen("test.out", "w", stdout);
#endif
#ifndef LOCAL
  freopen("P1527_[National_Team]_Matrix_Multiplication.in", "r",
          stdin);
  freopen("P1527_[National_Team]_Matrix_Multiplication.out", "w",
          stdout);
#endif
#endif
  STDquantum::main();
  return 0;
}
```

# Dynamic Rankings

[P2617 Dynamic Rankings](https://www.luogu.com.cn/problem/P2617)

## 题目描述

给定一个含有 $n$ 个数的序列 $$a_1,a_2 \dots a_n$$，需要支持两种操作： 

- `Q l r k` 表示查询下标在区间 $[l,r]$ 中的第 $k$ 小的数 

- `C x y` 表示将 $a_x$ 改为 $y$

## 输入输出格式

### 输入格式

第一行两个正整数 $n,m$，表示序列长度与操作个数。 第二行 $n$ 个整数，表示 $$a_1,a_2 \dots a_n$$。 接下来 $m$ 行，每行表示一个操作，都为上述两种中的一个。

### 输出格式

对于每一次询问，输出一行一个整数表示答案。

## 输入输出样例

### 输入样例 #1

```
5 3
3 2 1 4 7
Q 1 4 3
C 2 6
Q 2 5 3
```

### 输出样例 #1

```
3
6
```

## 说明

【数据范围】 

对于 $10\%$ 的数据，$1\le n,m \le 100$；

对于 $20\%$ 的数据，$1\le n,m \le 1000$；

对于 $50\%$ 的数据，$1\le n,m \le 10^4$；

对于 $100\%$ 的数据，$1\le n,m \le 10^5$，$1 \le l \le r \le n$，$1 \le k \le r-l+1$，$1\le x \le n$，$0 \le a_i,y \le 10^9$。 

请注意常数优化，但写法正常的整体二分和树套树都可以以大约 $1000\text{ms}$ 每个点的时间通过。 

来源：bzoj1901 

本题数据为洛谷自造数据，使用CYaRon耗时5分钟完成数据制作。

## 题解

这题就有一些可持久化的意味了。

然鹅整体二分的话，需要小小地转化一下视角。

我们看之前插入的地方的第$k$小就好了qwq。

你可能就会问了，更改和查询在一块不就乱套了吗？？？（~~可能你们都懂了只有我在问/kk~~

对于更改的就在原位上减去原来的值，再加上要改的值就好了。

因为我们在操作的时候是按照时间顺序插入的，遍历也是时间顺序，分到每一组后还是时间顺序，这就为我们的操作提供了正确性。

好的口胡完毕（并不是证明）直接套板子就好了QAQ

```cpp
#include <bits/stdc++.h>
#define LOCAL
using namespace std;

namespace STDquantum {

const int N = 6e5 + 10;
int n, m, cnt, cntq, ans[N], a[N];

struct Tree {
#define lowbit(x) ((x) & (-x))
  int t[N];
  inline void add(int x, int w) {
    while (x <= n) { t[x] += w, x += lowbit(x); }
  }
  inline int query(int x) {
    int tmp = 0;
    while (x) { tmp += t[x], x -= lowbit(x); }
    return tmp;
  }
  inline int sum(int x, int y) {
    return query(y) - query(x - 1);
  }
#undef lowbit
} T;

struct Q {
  int l, r, k, id, op;
} q[N], q1[N], q2[N];
inline void solve(int l, int r, int L, int R) {
  if (L > R) return;
  if (l == r) {
    for (int i = L; i <= R; ++i) {
      if (q[i].op == 2) ans[q[i].id] = l;
    }
    return;
  }
  int mid = (l + r) >> 1, cnt1 = 0, cnt2 = 0;
  for (int i = L; i <= R; ++i) {
    if (q[i].op == 1) {
      if (q[i].l <= mid) {
        q1[++cnt1] = q[i], T.add(q[i].id, q[i].r);
      } else {
        q2[++cnt2] = q[i];
      }
    } else {
      int tmp = T.sum(q[i].l, q[i].r);
      if (q[i].k <= tmp) {
        q1[++cnt1] = q[i];
      } else {
        q[i].k -= tmp, q2[++cnt2] = q[i];
      }
    }
  }
  for (int i = 1; i <= cnt1; ++i) {
    if (q1[i].op == 1) T.add(q1[i].id, -q1[i].r);
  }
  for (int i = 1; i <= cnt1; ++i) { q[L + i - 1] = q1[i]; }
  for (int i = 1; i <= cnt2; ++i) {
    q[L + cnt1 - 1 + i] = q2[i];
  }
  solve(l, mid, L, L + cnt1 - 1);
  solve(mid + 1, r, L + cnt1, R);
}
inline void main() {
  scanf("%d%d", &n, &m);
  int maxx = 0, minn = 0x3f3f3f3f;
  for (int i = 1; i <= n; ++i) {
    scanf("%d", a + i);
    maxx = max(maxx, a[i]), minn = min(minn, a[i]);
    q[++cnt] = (Q){a[i], 1, 0, i, 1};
  }
  char c[10];
  for (int i = 1, x, y, z; i <= m; ++i) {
    scanf("%s%d%d", c, &x, &y);
    if (c[0] == 'Q') {
      scanf("%d", &z);
      q[++cnt] = (Q){x, y, z, ++cntq, 2};
    } else {
      maxx = max(maxx, y), minn = min(minn, y);
      q[++cnt] = (Q){a[x], -1, 0, x, 1}; // -1代表要把这个数删去
      q[++cnt] = (Q){a[x] = y, 1, 0, x, 1}; // 1就给他把y加回去了qwq
    }
  }
  solve(minn, maxx, 1, cnt);
  for (int i = 1; i <= cntq; ++i) { printf("%d\n", ans[i]); }
}

}  // namespace STDquantum

int main() {
#ifndef ONLINE_JUDGE
#ifdef LOCAL
  freopen("test.in", "r", stdin);
  freopen("test.out", "w", stdout);
#endif
#ifndef LOCAL
  freopen("test.in", "r", stdin);
  freopen("test.out", "w", stdout);
#endif
#endif
  STDquantum::main();
  return 0;
}
```

# [ZJOI2013]K大数查询

[P3332 [ZJOI2013]K大数查询](https://www.luogu.com.cn/problem/P3332)

## 题目描述

你需要维护 $n$ 个可重整数集，集合的编号从 $1$ 到 $n$。

这些集合初始都是空集，有 $m$ 个操作：

- `1 l r c`：表示将 $c$ 加入到编号在 $[l,r]$ 内的集合中
- `2 l r c`：表示查询编号在 $[l,r]$ 内的集合的并集中，第 $c$ 大的数是多少。 

注意可重集的并是不去除重复元素的，如 $$\{1,1,4\}\cup\{5,1,4\}=\{1,1,4,5,1,4\}$$。

## 输入输出格式

### 输入格式

第一行两个正整数 $n,m$，表示集合个数和操作个数。

接下来 $m$ 行，每行四个整数表示一次操作。

### 输出格式

对于每个 $2$ 操作，输出一行一个整数表示答案。

## 输入输出样例

### 输入样例 #1

```
2 5
1 1 2 1
1 1 2 2
2 1 1 2
2 1 1 1
2 1 2 3
```

### 输出样例 #1

```
1
2
1
```

## 说明

【样例说明】 

第 $1$ 次操作在 $1,2$ 号集合中分别加入了一个 $1$。

第 $2$ 次操作在 $1,2$ 号集合中分别加入了一个 $2$。

第 $3$ 次操作查询 $1$ 号集合中第 $2$ 大的数，答案为 $1$。

第 $4$ 次操作查询 $1$ 号集合中第 $1$ 大的数，答案为 $2$。

第 $5$ 次操作查询 $1,2$ 号集合的并集 $$\{1,2,1,2\}$$ 中第 $3$ 大的数，答案为 $1$。 

【数据范围】

 $1 \le n,m \le 5\times 10^4$

 $1\le l,r \le n$

$1$ 操作中 $|c|\le n$ 

$2$ 操作中 $1\le c < 2^{63}$

## 题解

整体二分其实就是建出模型来，然后把板子一套。

所以首先我们寻找这道题和之前做过的题的不同之处，我们发现这个题查询的是区间第 $k$ 大，而不是小。我们先考虑这个操作如何实现。

回想我们前面写过的代码中，在插入操作的时候是当被加数小于$mid$的时候来进行插入操作，这样做的原因就是可以统计所有小于等于$mid$的数是不是够$k$个，以便决定查询区间是分到左边还是右边。这样，聪明的你是不是已经猜出来第$k$大怎么做了呢？

没错！就是把所有大于$mid$的数放到树状数组里查询就好了。

好的，解决了一个问题，我们还可以发现以前做出的修改查询什么的都是单点操作，那么区间操作呢？

线段树？不用。依旧是我们熟悉的树状数组。

那么怎么操作呢？

引入差分概念：

$c[i]=a[i]-a[i-1]$

那么我对某个区间内部$+K$实际上等价于区间内部所有$c[i]+K$

那么某个元素的值其实等于$a[i]=\sum_1^nc[i]$

那么区间和呢？

我们可以知道

$$
\begin{align}
\sum_1^na[i]
&=a[1]+a[2]+a[3]+\times+a[n]\\

&=a[1]+a[2]+a[3]+\times+a[n]\\

&=c[1]+c[1]+c[2]+c[1]+c[2]+c[3]+\cdots+c[1]+c[2]+\cdots+c[n-1]+c[n]\\

&=n\times(c[1]+c[2]+\cdots+c[n])-(c[2]+c[3]+\cdots+c[n]+c[3]+\cdots+c[n]+\cdots+c[n]+c[n])\\

&=n\cdot \sum _{1}^nc[i]-\sum _1^nc[i]\times(i-1)\\

&=\sum _1^nc[i]\times(n-i+1)\end{align}
$$

因此我们需要维护两个值：差分数组$c[i]=a[i]-a[i-1]$和$b[i]=c[i]\times(i-1)$

### 单点更新：

$c[i]$还是与原来的树状数组一样更新，$c[i]+x$即可

$b[i]=c[i]\times(i-1)$，那么$b[i]=(c[i]+x)\times(i-1)$，即$b[i]=c[i]\times(i-1)+x\times(i-1)$。我们更新$x\times(i-1)$即可。

更新效果：把$x$位置后面所有的数的值$+w$

### 区间更新：

$[L,R]+w=\mbox{update}(r,w)-\mbox{update}(l-1,w)$

更新效果：把$l$位置到$r$位置所有的数的值$+w$

### 区间求和：

由上面证明可知:

$$\sum_1^n a[i] = n\times\sum_1^n c[i] - \sum_1^n b[i] $$

和以前一样求和即可，求和式子换成上面那种即可

更新效果：$\mbox{sum}(x)=\sum_1^x a[i]$

```cpp
#include <bits/stdc++.h>
#define LOCAL
using namespace std;

namespace STDquantum {

typedef long long ll;
const ll N = 6e5 + 10;
ll n, m, cnt, cntq, ans[N];

struct T {
  ll sum1[N], sum2[N];
  inline void add(ll x, ll w) {
    for (ll i = x; i <= n; i += i & -i) {
      sum1[i] += w, sum2[i] += w * x; // 其实是等价的写法，和query对应
    }
  }
  inline void modify(ll l, ll r, ll w) {
    add(l, w), add(r + 1, -w);
  }
  inline ll query(ll x) {
    ll tmp = 0;
    for (ll i = x; i; i -= i & -i) {
      tmp += (x + 1) * sum1[i] - sum2[i];
    }
    return tmp;
  }
  inline ll sum(ll l, ll r) { return query(r) - query(l - 1); }
} T;

struct Q {
  ll l, r, k, id, op;
} q[N], q1[N], q2[N];
inline void solve(ll l, ll r, ll L, ll R) {
  if (L > R) return;
  if (l == r) {
    for (ll i = L; i <= R; ++i) {
      if (q[i].op == 2) ans[q[i].id] = l;
    }
    return;
  }
  ll mid = (l + r) >> 1, cnt1 = 0, cnt2 = 0;
  for (ll i = L; i <= R; ++i) {
    if (q[i].op == 1) {
      if (q[i].k > mid) { // 变化在这里！
        q2[++cnt2] = q[i], T.modify(q[i].l, q[i].r, 1); // 区间加个1
      } else {
        q1[++cnt1] = q[i];
      }
    } else {
      ll tmp = T.sum(q[i].l, q[i].r);
      if (q[i].k <= tmp) {
        q2[++cnt2] = q[i];
      } else {
        q[i].k -= tmp, q1[++cnt1] = q[i];
      }
    }
  }
  for (ll i = 1; i <= cnt2; ++i) {
    if (q2[i].op == 1) T.modify(q2[i].l, q2[i].r, -1);
  }
  for (ll i = 1; i <= cnt1; ++i) { q[L + i - 1] = q1[i]; }
  for (ll i = 1; i <= cnt2; ++i) { q[L + cnt1 - 1 + i] = q2[i]; }
  solve(l, mid, L, L + cnt1 - 1);
  solve(mid + 1, r, L + cnt1, R);
}
inline void main() {
  scanf("%lld%lld", &n, &m);
  ll maxx = -0x7fffffff, minn = 0x7fffffff;
  for (ll i = 1, op, l, r, c; i <= m; ++i) {
    scanf("%lld%lld%lld%lld", &op, &l, &r, &c);
    maxx = max(maxx, c), minn = min(minn, c);
    if (op == 1) {
      q[++cnt] = (Q){l, r, c, i, op};
    } else {
      q[++cnt] = (Q){l, r, c, ++cntq, op};
    }
  }
  solve(minn, maxx, 1, cnt);
  for (ll i = 1; i <= cntq; ++i) { printf("%lld\n", ans[i]); }
}

}  // namespace STDquantum
```
