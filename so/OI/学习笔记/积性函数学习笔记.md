# 常见积性函数

## 积性函数定义

积性函数是指一个定义域为 $\mathbb{N}^+$ 的算术函数 $f(n)$ ， 有如下性质：$f(1)=1$，且对于 $\forall a,b∈\mathbb{N}^+$ 且 $\gcd(a,b)=1$ ，有 $f(ab)=f(a)f(b)$。

若对于任意的 $a,b$， $f$ 均满足上述性质，则称此函数为完全积性函数。

## 常见积性函数

- $\displaystyle \varphi(n)=\sum\limits_{i=1}^{n}[\gcd(i,n)=1]$，欧拉函数。

  欧拉函数还有一个求法，$n=p_1^{k_1}p_2^{k_2}\dots p_r^{k_r}$，则 $\displaystyle\varphi(n)=n\sum_{i=1}^r1-\frac 1{p_i}=n\sum_{i=1}^r\frac {p_i-1}{p_i}$，从这里我们可以推出 $\displaystyle \varphi(n^m)=n^m\sum_{i=1}^r\frac {p_i-1}{p_i}=n^{m-1}\varphi(n)$。

  对于 $\varphi(n^m)$ 给出定义域 $\left\{n^m\middle|n\in\mathbb N^+\right\}$，$\varphi(n^m)$ 在定义域内是积性函数。

- $\displaystyle \mu(n)=\left\{\begin{array}{ll}1 & n=1\\ (-1)^k & n=p_1p_2\dots p_k\\0 & {\rm else}\end{array}\right.=\begin{cases}1 & n=1 \\ 0 & \exists d>1:d^{2} \mid n \\ (-1)^{\omega(n)} & {\rm otherwise}\end{cases}$，莫比乌斯函数。

- $\displaystyle\omega(n)=\sum_{i=1}^n[i\mid n][i\in\text{prime}]$，不同质因子个数。

- $\gcd(n,k)$，$k$ 是常数。

- $σ_k(n)$ —— 除数函数，$n$ 的所有正因数的 $k$ 次幂之和，当中 $k$ 可为任何复数。在特例中有：

  $σ_0(n)=d(n)$ —— $n$ 的正因数数目。

  $σ_1(n)=σ(n)$ —— $n$ 的所有正因数之和。

- $\text{id}_k(n)=n^k$——幂函数，在特例中有：

  $\text{id}_0(n)=1(n)=1$，恒为 $1$ 的常数函数。

  $\text{id}_1(n)=\text{id}(n)=n$。

- $\varepsilon(n)=[n=1]$，单位元函数。

## 积性函数证明技巧

当想要证明一个函数是积性函数时，如 $\displaystyle\text{F}(n)=\sum_{d\mid n}d\cdot\mu(d)$。

可以考虑 $a,b\in[1,n],a\perp b$（垂直符号表示 $a,b$ 互质），$\displaystyle \text{F}(a)=\sum_{d\mid a}d\cdot\mu(d),\,\text{F}(b)=\sum_{d\mid b}d\cdot\mu(d)$。

可以发现在 $\displaystyle\text{F}(a\cdot b)=\sum_{d\mid a\cdot b}d\cdot\mu(d)$ 中，每一项 $d$ 都是由 $a$ 和 $b$ 分别的贡献的积组成的，如 $d=ab$ 中，$a\cdot b\cdot \mu(a\cdot b)=a\cdot\mu(a)\cdot b\cdot \mu(b)$。

所以就有 $\text{F}$ 为积性函数。

## 积性函数筛法

### 欧拉函数

```cpp
int num, phi[N], prime[N];
bool isPrime[N];
inline void Get(int n) {
  memset(isPrime, 1, sizeof(isPrime));
  isPrime[1] = 0, phi[1] = 1;
  for (int i = 2; i <= n; ++i) {
    if (isPrime[i]) prime[++num] = i, phi[i] = i - 1;
    for (int j = 1; j <= num && i * prime[j] <= n; ++j) {
      isPrime[i * prime[j]] = 0;
      if (i % prime[j] == 0) {
        phi[i * prime[j]] = phi[i] * prime[j];
        break;
      }
      phi[i * prime[j]] = phi[i] * phi[prime[j]];
    }
  }
}
```

### 莫比乌斯函数

```cpp
int num, prime[N], mu[N];
bool isPrime[N];
inline void Get(int n) {
  memset(isPrime, 1, sizeof(isPrime));
  isPrime[1] = 0, mu[1] = 1;
  for (int i = 2; i <= n; ++i) {
    if (isPrime[i]) prime[++num] = i, mu[i] = -1;
    for (int j = 1; j <= num && i * prime[j] <= n; ++j) {
      isPrime[i * prime[j]] = 0;
      if (i % prime[j] == 0) {
        mu[i * prime[j]] = 0;
        break;
      }
      mu[i * prime[j]] = -mu[i];
    }
  }
}
```

### 一般积性函数

来自`自为风月马前卒`大佬（orz%%%）

它有常见的套路，假设 $n = p_1^{a_1} p_2^{a_2} \dots p_k^{a_k}$，如果我们能快速得出 $f(p_i)$ 和 $f(p_i^{k+1})$ 的取值，那么直接套板子就行了。

为了方便筛，我们需要把问题无限简化，设 $low(i)$ 表示 $p_1^{a_1}$，考虑筛法中最关键的地方$i\equiv0\pmod{p_j}$，有了 $low(i)$，此时我们需要分两种情况讨论：

1. $low(i) = i$，此时 $i$ 一定是某个素数的幂的形式（否则就会break掉）。

   这里就用到了我最开始说的那个套路。

   如果我们能快速的利用 $f(p_i^{k})$ 更新出 $f(p_i^{k + 1})$，那这个素数就很容易筛了。

2. $low(i) \not = i$，那么 $\displaystyle\frac i{low(i)}$ 一定与 $low(i) \cdot p_j$ 是互质的，我们可以直接利用积性函数的性质去更新。

伪代码（`ll`是防溢出的）：

```cpp
int num, prime[N];
ll low[N], H[N];
bool isPrime[N];
inline void Get(int n) {
  memset(isPrime, 1, sizeof(isPrime));
  isPrime[1] = 0, low[1] = H[1] = 1;
  for (int i = 2; i <= n; ++i) {
    if (isPrime[i])
      prime[++num] = i, H[i] = 素数时的答案, low[i] = i;
    for (int j = 1, k; j <= num && (ll)i * prime[j] <= n; ++j) {
      isPrime[k = i * prime[j]] = 0;
      if (i % prime[j] == 0) {
        low[k] = low[i] * prime[j];
        if (low[i] == i) {
          H[k] = 从H[i]转化来;
        } else {
          H[k] = H[i / low[i]] * H[prime[j] * low[i]]; // 积性函数性质
        }
        break;
      }
      H[k] = H[i] * H[prime[j]];
      low[k] = prime[j];
    }
  }
}
```

# 狄利克雷卷积

狄利克雷卷积——数论函数最重要的操作。

用 $\ast $ 表示，定义为 $\displaystyle(f\ast g)(n)=\sum_{d\mid n}f(d)g(\frac nd)=\sum_{ab=n}f(a)g(b)$。

积性函数对于狄利克雷卷积是封闭的（积性函数卷积性函数还是积性函数）。

- $\varepsilon\ast f=f$

- $\displaystyle d(n)=\sum\limits_{d\mid n}1$，即 $d=1\ast 1$。
- $\displaystyle \sigma(n)=\sum\limits_{d\mid n}d$，即 $\sigma=1\ast \text{id}$。
- $\displaystyle \varphi(n)=\sum\limits_{d\mid n}\mu(d)\frac nd$，即 $\varphi=\text{id}\ast \mu$

- $\displaystyle \varepsilon(n)=\sum\limits_{d\mid n}\mu(d)$，即 $\varepsilon=1\ast \mu$。
- $1\ast \varphi=1\ast \text{id}\ast \mu=(1\ast \mu)\ast \text{id}=\varepsilon\ast \text{id}=\text{id}$，即 $1\ast \varphi=\text{id}$

由 $\varepsilon=1\ast \mu$ 可得到莫比乌斯反演常用公式 $f=1\ast g\Rightarrow f\ast \mu=1\ast g\ast \mu=g$
