# P2473 [SCOI2008] 奖励关

## 题目描述

你正在玩你最喜欢的电子游戏，并且刚刚进入一个奖励关。在这个奖励关里，系统将依次随机抛出 $k$ 次宝物，每次你都可以选择吃或者不吃（必须在抛出下一个宝物之前做出选择，且现在决定不吃的宝物以后也不能再吃）。

宝物一共有 $n$ 种，系统每次抛出这 $n$ 种宝物的概率都相同且相互独立。也就是说，即使前 $(k-1)$ 次系统都抛出宝物 $1$（这种情况是有可能出现的，尽管概率非常小），第 $k$ 次抛出各个宝物的概率依然均为 $\frac 1 n $。 

获取第 $i$ 种宝物将得到 $p_i$ 分，但并不是每种宝物都是可以随意获取的。第 $i$ 种宝物有一个前提宝物集合 $s_i$。只有当 $s_i$ 中所有宝物都至少吃过一次，才能吃第 $i$ 种宝物（如果系统抛出了一个目前不能吃的宝物，相当于白白的损失了一次机会）。注意，$p_i$ 可以是负数，但如果它是很多高分宝物的前提，损失短期利益而吃掉这个负分宝物将获得更大的长期利益。

假设你采取最优策略，平均情况你一共能在奖励关得到多少分值？

## 输入输出格式

### 输入格式

第一行为两个整数，分别表示抛出宝物的次数 $k$ 和宝物的种类数 $n$。 

第 $2$ 到第 $(n + 1)$ 行，第 $(i + 1)$ 有若干个整数表示第 $i$ 个宝物的信息。每行首先有一个整数，表示第 $i$ 个宝物的分数 $p_i$。接下来若干个互不相同的整数，表示该宝物的各个前提宝物集合 $s_i$，每行的结尾是一个整数 $0$，表示该行结束。

### 输出格式

输出一行一个实数表示答案，保留六位小数。

## 输入输出样例

### 输入样例 #1

```
1 2
1 0
2 0
```

### 输出样例 #1

```
1.500000
```

### 输入样例 #2

```
6 6
12 2 3 4 5 0
15 5 0
-2 2 4 5 0
-11 2 5 0
5 0
1 2 4 5 0
```

### 输出样例 #2

```
10.023470
```

## 说明

#### 数据规模与约定 

对于全部的测试点，保证 $1 \leq k \leq 100$，$1 \leq n \leq 15$，$-10^6 \leq p_i \leq 10^6$。

# 做题


上课讲的一道期望DP。

写出来感觉很简单，但是因为不太熟，推导的过程很困难。不看题解还是做不对T_T

看到$n$的范围会想到状态压缩，正好也是可以考虑选与不选某个宝物。

我们发现，这个题如果正向推的话非常费劲，状态很多，没法转移，还有后效性。所以考虑状态很少的逆序推。

在推的时候，可以设置$f[i][s]$状态，$i$表示第几轮，$s$是状态压缩的一个总体状态，一位上是$1$表示选过这个物品，$0$则相反。这样转移的时候要涉及到一个$S[i]$集合表示之前必须选什么，就可以$(S[i]\&s)==S[i]$判断是不是前缀集合都选了。

然后就枚举判断一个$l$，看$S[l]$是否满足条件。

如果满足，那么可以选择到底选不选这个物品，最优策略就取$\max$就可。

如果不满足，那么直接继承。

然后就看代码吧。


```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

// 蓝蓝的天空银河里，有只小白船~
// 船上有棵桂花树，白兔在游玩~

const int N = 16;
int k, n, S[N];
double f[120][1 << N], w[N];
inline void main() {
	scanf("%d%d", &k, &n);
	for(int i = 1, x; i <= n; ++i) {
		scanf("%lf", w + i);
		while (~scanf("%d", &x) && x) {
			S[i] |= 1 << x - 1;
		}
	}
	for(int i = k; i; --i) {
		for(int j = (1 << n) - 1; ~j; --j) {
			for(int l = 1; l <= n; ++l) {
				if ((S[l] & j) == S[l]) {
					f[i][j] += max(f[i + 1][j], f[i + 1][j | (1 << l - 1)] + w[l]);
				} else {
					f[i][j] += f[i + 1][j];
				}
			}
			f[i][j] /= n;
		}
	}
	printf("%.6lf", f[1][0]);
}

} // namespace STDquantum

int main() {
	STDquantum::main();
	return 0;
}
```
