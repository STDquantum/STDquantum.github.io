# P2602 [ZJOI2010]数字计数

## 题目描述

给定两个正整数 $a$ 和 $b$，求在 $[a,b]$ 中的所有整数中，每个数码(digit)各出现了多少次。

## 输入输出格式

### 输入格式

仅包含一行两个整数 $a,b$，含义如上所述。

### 输出格式

包含一行十个整数，分别表示 $0\sim 9$ 在 $[a,b]$ 中出现了多少次。

## 输入输出样例

### 输入样例 #1

```
1 99
```

### 输出样例 #1

```
9 20 20 20 20 20 20 20 20 20
```

## 说明

#### 数据规模与约定 

- 对于 $30\%$ 的数据，保证 $a\le b\le10^6$；

- 对于 $100\%$ 的数据，保证 $1\le a\le b\le 10^{12}$。

# 感想

这题看上去不难，其实状态比较不容易推出来，作为一道比较经典的数位DP应用，它作为紫题自有它的思维难度所在。那就是**转化条件**。

我们要求数位上的东西，但是哪个位数也不一样多，所以逐个求解异常困难。所以既然不考虑前导零，那么不妨考虑上前导零，这样看起来不是有规律很多了吗（不是

因为相同位数的看起来会很整齐，所以对于同样位数的一些数，比如位数为$4$，它的范围是$0000-9999$，在这个范围内，所有十个数字的个数同样多！那这样我们就可以简单地合并一下，令$f[i]$为位数为$i$的所有数字中其中一个数字的个数。那么转移就需要动动脑筋，首先多了一位数，之前的无论什么都原封不动重复$10$次（因为有前导零），再加上作为新增添的一位每次多了$10^{i-1}$个，$f[i]=f[i-1]*10+10^{i-1}$。这个要是不懂的话可以写一写从一位数到两位数的转化考虑，就会明白了。

然后就是怎么考虑算出答案了。首先对于$00\dots00-A0\dots00$（$A$表示任意一个数字）中，相当于去掉首位后剩下部分重复了$A$次，从$0$到$A-1$，把这部分累加进答案。然后这些首位呢又重复了$10$的位数减一次方次，也就是比如四位数就重复了$10^3=1000$次，把这部分也累加进答案。这时候我们会发现如果对于$AB\dots CD$的形式，剩下部分就是$A$重复了$B\dots CD+1$次（加一是因为还有$A0\dots00$），然后再累加上$B\dots CD$的答案，于是我们就发现成功计算了一个数位对答案的贡献。接下来递归操作就可以了。

你可能会问了，前导零呢？其实前导零的影响非常好去除，你只需要在每次进入下一次递归的时候减去$10^{\mbox{位数-1}}$就可以了，和前面算$f[i]$时候的$10^{i-1}$是一个道理。

挂代码

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

const int N = 20;
long long a, b, f[N], t[N], cnta[N], cntb[N];
void init(long long n) {
	t[0] = 1;
	for(int i = 1; i <= n; ++i) {
		t[i] = t[i - 1] * 10; // 预处理10的幂
	}
}
void solve(long long y, long long *cnt) { // 省得写两个函数，传参方便
	long long num[N], len = 0, x = y;
	while (x) {
		num[++len] = x % 10;
		x /= 10;
	}
	for(long long i = 1; i <= len; ++i) {
		f[i] = f[i - 1] * 10.0 + t[i - 1];
	}
	for(long long i = len; i; --i) { // （假）递归过程
		for(long long k = 0; k < 10; ++k) {
			cnt[k] += f[i - 1] * num[i];
		}
		for(long long k = 0; k < num[i]; ++k) {
			cnt[k] += t[i - 1];
		}
		cnt[num[i]] += y % t[i - 1] + 1;
		cnt[0] -= t[i - 1]; // 去除前导零影响
	}
}
inline void main() {
	scanf("%lld%lld", &a, &b);
	init(14);
	solve(b, cntb);
	solve(a - 1, cnta);
	for(long long i = 0; i <= 9; ++i) {
		printf("%lld ", cntb[i] - cnta[i]); // 前缀和相减是非常常用的区间算法
	}
}

} // namespace STDquantum

int main() {
	STDquantum::main();
	return 0;
}
```
