# P3195 [HNOI2008]玩具装箱

## 题目描述

P 教授要去看奥运，但是他舍不下他的玩具，于是他决定把所有的玩具运到北京。他使用自己的压缩器进行压缩，其可以将任意物品变成一堆，再放到一种特殊的一维容器中。 

P 教授有编号为 $1 \cdots n$ 的 $n$ 件玩具，第 $i$ 件玩具经过压缩后的一维长度为 $C_i$。 

为了方便整理，P教授要求： 

- 在一个一维容器中的玩具编号是连续的。 
- 同时如果一个一维容器中有多个玩具，那么两件玩具之间要加入一个单位长度的填充物。形式地说，如果将第 $i$ 件玩具到第 $j$ 个玩具放到一个容器中，那么容器的长度将为 $x=j-i+\sum\limits_{k=i}^{j}C_k$。 

制作容器的费用与容器的长度有关，根据教授研究，如果容器长度为 $x$，其制作费用为 $(x-L)^2$。其中 $L$ 是一个常量。P 教授不关心容器的数目，他可以制作出任意长度的容器，甚至超过 $L$。但他希望所有容器的总费用最小。

## 输入输出格式

### 输入格式

第一行有两个整数，用一个空格隔开，分别代表 $n$ 和 $L$。 

第 $2$ 到 第 $(n + 1)$ 行，每行一个整数，第 $(i + 1)$ 行的整数代表第 $i$ 件玩具的长度 $C_i$。

### 输出格式

输出一行一个整数，代表所有容器的总费用最小是多少。

## 输入输出样例

### 输入样例 #1

```
5 4
3
4
2
1
4
```

### 输出样例 #1

```
1
```

## 说明

对于全部的测试点，$1 \leq n \leq 5 \times 10^4$，$1 \leq L \leq 10^7$，$1 \leq C_i \leq 10^7$。

# 做题

怎么说呢，刚讲了斜率优化DP，之前听过几次，原理我懂，可是不会写。

这次终于整了点阳间东西。

经过读题，可以列出一个比较简单的DP

设$f[i]$为到第$i$个物品的最小代价，那么怎么转移呢？可以考虑一个在$[1,i-1]$范围内的$j$，然后把$[j+1,i]$的物品放进一个容器。设物品长度的前缀和为$sum[]$，这样转移为
$$
f[i]=\min_{1\le j\le i}\left(f[j]+(i-(j+1)+sum[i]-sum[j]-L)^2\right)
$$
我们可以发现这个枚举是$\mathcal{O}(n^2)$的，根本不足以通过5e4的数据，那么我们可以考虑用一下斜率优化。

首先先去掉$\min$，因为肯定不能这么枚举。为了简化问题，我们把$sum[i]+i+L+1$叫做$b[i]$，把$sum[i]+i$叫做$a[i]$。于是可以化简原式
$$
f[i]=f[j]+(a[i]-b[j])^2\\=f[j]+a[i]^2-2\times a[i]\times b[j]+b[j]^2
$$
因为$i$是必须要从$1$到$n$枚举的，所以在求解$f[i]$时$i$是固定的，那么所有带$i$的都成了只和$i$有关的“常数”。虽然如此，但并不是完全的常数，还是会随$i$变化的。而$j$是个完全的变量。

{% asset_image 1.png %}

把带$i$和$j$的项作为$kx$，只带$i$的项作为$b$，只带$j$的项作为$y$。这样需要决策的就变成了一根直线。

如上图，我们要让决策优一些，那么就要像线性规划一样，因为斜率是正的，所以是把直线往上挪动。于是要维护一个下凸包。

用单调队列来维护斜率就行。

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

const int N = 5e4 + 10;
int n, L, head, tail, q[N];
double sum[N], f[N];
inline double a(int i) { return 0.0 + sum[i] + i; }
inline double b(int i) { return 1.0 + a(i) + L; }
inline double X(int i) { return b(i); }
inline double Y(int i) { return f[i] + b(i) * b(i); }
inline double slope(int i, int j) {
  return (Y(i) - Y(j)) / (X(i) - X(j));
}
inline void main() {
  scanf("%d%d", &n, &L);
  for (int i = 1; i <= n; ++i) {
    scanf("%lf", &sum[i]);
    sum[i] += sum[i - 1];
  }
  head = tail = 1;
  for (int i = 1; i <= n; ++i) {
    while (head < tail && slope(q[head], q[head + 1]) < 2 * a(i)) {
      ++head;
    }
    f[i] = f[q[head]] + (a(i) - b(q[head])) * (a(i) - b(q[head]));
    while (head < tail &&
           slope(q[tail - 1], q[tail]) > slope(i, q[tail - 1])) {
      --tail;
    }
    q[++tail] = i;
  }
  printf("%lld", (long long)f[n]);
}

}  // namespace STDquantum

int main() {
#ifndef ONLINE_JUDGE

  freopen("P3195_[HNOI2008]_Toys.out", "w", stdout);
#endif

  STDquantum::main();
  return 0;
}
```
