# P4127 [AHOI2009]同类分布

## 题目描述

给出两个数$a,b$，求出$[a,b]$中各位数字之和能整除原数的数的个数。

## 输入输出格式

### 输入格式

一行，两个整数$a$和$b$

### 输出格式

一个整数，表示答案

## 输入输出样例

### 输入样例 #1

```
10 19
```

### 输出样例 #1

```
3
```

## 说明

对于所有的数据，$1 ≤ a ≤ b ≤ 10^{18}$

# 做题

放个代码：

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

typedef long long ll;
typedef bool bl;
const ll N = 20;
ll n, m, len, num[N], mod, f[N][170][170];
ll solve1(ll now, ll sum, bl xn, ll st) { // 分别为：第几位，各数位和，是否有取值限制，这个数
	if (!now) { // 结束条件：递归完了一个数
		if (!sum) return 0; // 0不符合条件
		return !st && sum == mod; // 如果模完了mod的st为零，而且各数位的和为mod，就找到一个
	}
	if (!xn && ~f[now][sum][st]) {
		return f[now][sum][st];
	}
	ll tmp = 0, lim = xn ? num[now] : 9, i;
	for(i = 0; i <= lim; ++i) {
		tmp += solve1(now - 1, sum + i, i == lim && xn, (st * 10 + i) % mod);
	}
	if (!xn) f[now][sum][st] = tmp;
	return tmp;
}
ll solve2(ll x) {
	len = 0;
	ll ans = 0;
	while (x) {
		num[++len] = x % 10;
		x /= 10;
	}
	for(mod = 1; mod <= 9 * len; ++mod) {
		memset(f, -1, sizeof(f));
		ans += solve1(len, 0, 1, 0); // 起始为len，总和是0，这位不能随便取，这数现在是0
	}
	return ans;
}
inline void main() {
	scanf("%lld%lld", &n, &m);
	printf("%lld", solve2(m) - solve2(n - 1));
}

} // namespace STDquantum

int main() {
	STDquantum::main();
	return 0;
}
```
