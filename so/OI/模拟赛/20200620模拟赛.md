# 连珠风暴

（necklace.pas/c/cpp） 

## **问题描述：**

给定$M$种颜色的珠子，每种颜色珠子的个数均不限，将这些珠子做成长度为$N$的项链。

问能做成多少种不重复的项链。

并且两条项链相同，当且仅当两条项链通过旋转或是翻转后能重合在一起，且对应珠子的颜色相同。

## **样例输入：**

```
2 5
```

## **样例输出：**

```
8
```

## **下图是样例解释：**

## **额外提供几组数据：**

| $m$  | $n$  | $ans$ |
| :--: | :--: | :---: |
|  2   |  1   |   2   |
|  2   |  2   |   3   |
|  5   |  1   |   5   |
|  2   |  6   |  13   |
|  6   |  2   |  21   |



## **数据范围：**

$30\%$：$n,m\le4$

$60\%$：$n,m\le5$

$100\%$：$nm\le32$

## **做法**

这是一道群论的模板题，要用到Polya定理。下面对于Polya定理的介绍引自百度百科：

> Pólya定理是非常重要和基本的计数工具。Pólya定理是匈牙利数学家Pólya利用发生函数的方法，结合群的观点和权的概念建立起来的一个有关计数定理。Pólya定理在有关计算不同等价类的个数问题上起着重要的作用。

考虑题目中给出的两种限制：旋转同构和翻转同构

下文中的循环节其实就是在一种映射下产生的环个数

1. 对于旋转同构，考虑旋转的珠子数

   可以知道若在$n$个珠子中旋转$i$位，那么其循环节是$\frac{n*i}{\mbox{lcm}(n,i)}=\gcd(n,i)$。

2. 对于翻转同构，要进行对$n$的奇偶性讨论。$n$为奇数的时候和为偶数的时候，结果是不同的。

   - $n$为奇数

     在$n$为奇数的时候，中心的珠子不变而将剩下珠子翻转，那么有$n$种翻转同构方式，每个翻转同构的循环节是$\frac{n+1}2$。

   - $n$为偶数

     $n$为偶数的时候，有两种翻转同构的方式。对称轴不同，决定了其翻转同构的不同。如下图，红色所示的对称方式，共有$\frac n2$组正对的珠子，则共有$\frac n2$种翻转同构方式，其循环节为$\frac{n+2}2$。蓝色所示的对称方式，共有$\frac n2$种翻转同构方式，其循环节为$\frac n2$。

     {% asset_image 2.png %}

OK讨论完毕，可以开始写代码了。

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

const int N = 1e6 + 10;
long long m, n, ans;
long long gcd(long long a, long long b) {
	return (b == 0) ? a : gcd(b, a % b);
}
inline void main() {
	scanf("%lld%lld", &m, &n);
	for(long long i = 1; i <= n; ++i) {
		ans += (long long)(pow(m, gcd(n, i)));
	}
	if (n % 2 == 1) {
		ans += n * (long long)(pow(m, (n + 1) / 2));
	} else {
		ans += (n / 2) * (long long)(pow(m, (n + 2) / 2));
		ans += (n / 2) * (long long)(pow(m, n / 2));
	}
	printf("%lld", ans / (2 * n)); // 别忘了除掉2n
}

} // namespace STDquantum

int main() {
	freopen("necklace.in", "r", stdin);
	freopen("necklace.out", "w", stdout);
	STDquantum::main();
	fclose(stdin), fclose(stdout);
	return 0;
}
```

#    道路修建

（road.pas/c/cpp）

## **问题描述：**

为了保护放牧环境，避免牲畜过度啃咬同一个地方的草皮，牧场主决定利用不断迁移牲畜进行喂养的方法去保护牧草。然而牲畜在迁移过程中也会啃食路上的牧草，所以如果每次迁移都用同一条道路，那么该条道路同样会被啃咬过度而遭受破坏。

现在牧场主拥有$F$个农场，已知这些农场至少有一条路径连接起来（不一定是直接相连），但从某些农场去另外一些农场，至少有一条路可通行。为了保护道路上的牧草，农场主希望再建造若干条道路，使得每次迁移牲畜时，至少有$2$种迁移途径，避免重复走上次迁移的道路。已知当前有的$R$条道路，问农场主至少要新建造几条道路，才能满足要求？

## **输入描述**

第一行$2$个正整数，分别为$n$和$m$

以下$m$行，每行$2$个数，表示连接的编号

## **输出描述：**

  一行一个数，表示至少新建的道路数目。

## **样例输入：**

```
7 7
1 2
2 3
3 4
2 5
4 5
5 6
5 7
```

## **样例输出：**

```
2
```

## **数据范围：**

$60\%$：$n\le100,m\le2000$

$100\%$：$n\le100000,m\le500000$

## 题解

用tarjan缩点后找出所有度数为$1$的点个数$x$，答案就是$\lfloor\frac{x+1}2\rfloor$。

下面是WanKuPi的代码（orz），把缩点完全学会了以后我就把我自己代码放上来。

```cpp
#include <cstdio>
void read(int &x) {
	x = 0; int c = getchar(), f = 1;
	while (c < '0' || c>'9') { if (c == '-') f = -1; c = getchar(); }
	while (c >= '0' && c <= '9') { x = x * 10 + c - '0'; c = getchar(); }
	x *= f;
}
int n = 0, m = 0;
int head[100001], nxt[1000007], to[1000007], cnt = 1;
inline void insert(int u, int e) { nxt[++cnt] = head[u]; head[u] = cnt; to[cnt] = e; }

int dfn[100001], low[100001], id[100001], c = 0;
int st[100001], u = 0;
bool in[100001];
inline int min(int a, int b) { return a < b ? a : b; }
void tarjan(int x, int li) {
	dfn[x] = low[x] = ++c;
	st[++u] = x;
	in[x] = true;
	for (int i = head[x]; i; i = nxt[i]) {
		if (i == li) continue;
		if (!dfn[to[i]]) {
			tarjan(to[i], i ^ 1);
			low[x] = min(low[x], low[to[i]]);
		}
		else if (in[to[i]])
			low[x] = min(low[x], dfn[to[i]]);
	}
	if (low[x] == dfn[x]) {
		int y = 0;
		do {
			y = st[u--];
			id[y] = x;
			in[y] = false;
		} while (y != x);
	}
}
int d[100001];
int main() {
	freopen("road.in", "r", stdin);
	freopen("road.out", "w", stdout);
	read(n); read(m);
	int a = 0, b = 0;
	for (int i = 1; i <= m; ++i) {
		read(a); read(b);
		insert(a, b);
		insert(b, a);
	}
	tarjan(1, 0);
	for (int x = 1; x <= n; ++x)
		for (int i = head[x]; i; i = nxt[i])
			if (id[to[i]] != id[x]) {
				++d[id[x]];
				++d[id[to[i]]];
			}
	int sum = 0;
	for (int x = 1; x <= n; ++x) if (dfn[x] == low[x] && d[x] == 2) ++sum;
	printf("%d", (sum + 1) / 2);
	fclose(stdin); fclose(stdout);
	return 0;
}
```

## **输入描述**

  第一行$2$个正整数，分别为$n$和$m$

  以下$m$行，每行$3$个数，表示边连接的信息，

## **输出描述：**

```
4 5
1 2 5
2 3 5
```

 ```
3.666667
 ```
