# matrix

## 题目描述

给定两个长度为 $n$ 的整数序列 $l$ 和 $t$ , 分别作为 $n$ x $n$ 矩阵 $F$ 的第一列和第一行，并且保证 $$l_1 = t_1$$。 同时矩阵中的任意其他元素 $$F_{i,j}$$ 由以下递推给定： 
$$
F_{i,j} = a\cdot F_{i,j-1} + b\cdot F_{i-1,j}
$$
给定系数$a,b,$ 要求计算$F_{n,n}$ 模 $10^9+7$ 的值。

## 输入输出格式

### 输入格式

第一行包含三个整数 $n,a,b$。 第二行包含$n$个整数$$l_i$$。 第三行包含$n$个整数$$t_i$$。

### 输出格式

共一行包含一个整数，表示$F_{n,n}$模$10^9+7$的值。

## 输入输出样例

### 输入样例 #1

```
4 3 5
4 1 7 3
4 7 4 8 
```

### 输出样例 #1

```
59716
```

## 说明

对于$40\%$的数据 ：$n \leq 5000$ 

对于另外$20\%$的数据 ：$a=0$ 

对于$100\%$的数据 ：$$n,a,b,l_i,t_i \leq 100000$$

然后推呀推呀，中间错了好几次，经过两个多小时的努力，终于把答案凑出来了，代码写出来了（好几处没加```1ll```调了好久）。

发现$$F_{1,1}$$没有什么用，于是从$$F_{2,1}$$到$$F_{n,1}$$编号为$$x_1$$到$$x_{n-1}$$，从$$F_{1,2}$$到$$F_{1,n}$$编号为$$y_1$$到$$y_{n-1}$$。
$$
ans=a^{n-1}\sum^{n-1}_{i=1}\left(\begin{array}c{2n-i-3}\\{n-2}\end{array}\right)b^{n-i-1}y_i+b^{n-1}\sum^{n-1}_{i=1}\left(\begin{array}c{2n-i-3}\\{n-2}\end{array}\right)a^{n-i-1}x_i
$$
我发誓我真的是生凑出来的。

代码：

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

const int N = 2e5 + 10, p = 1e9 + 7;
int n, inv[N], fac[N], a, b, x[N], y[N];
int fpm(int a1, int b1) {
	long long ans = 1;
	for(; b1; b1 >>= 1, a1 = 1ll * a1 * a1 % p) {
		if (b1 & 1) ans = ans * a1 % p;
	}
	return (int)(ans % p);
}
void init() {
	fac[0] = fac[1] = inv[0] = 1;
	for(int i = 2; i <= 2 * n; ++i) {
		fac[i] = 1ll * fac[i - 1] * i % p;
	}
	inv[2 * n] = fpm(fac[2 * n], p - 2);
	for(int i = 2 * n - 1; i; --i) {
		inv[i] = 1ll * inv[i + 1] * (i + 1) % p;
	}
}
int C(int n1, int m1) {
	return int((1ll * inv[n1 - m1] * inv[m1] % p) * fac[n1] % p);
}
inline void main() {
	scanf("%d%d%d", &n, &a, &b);
	for(int i = 0; i < n; ++i) {
		scanf("%d", &y[i]);
	}
	for(int i = 0; i < n; ++i) {
		scanf("%d", &x[i]);
	}
	init();
	long long tmp1 = 0, tmp2 = 0;
	for(int i = 1; i <= n - 1; ++i) {
		tmp1 = (tmp1 + (1ll * C(2 * n - i - 3, n - 2) * fpm(b, n - i - 1) % p) * y[i] % p) % p;
		tmp2 = (tmp2 + (1ll * C(2 * n - i - 3, n - 2) * fpm(a, n - i - 1) % p) * x[i] % p) % p;
	}
	tmp1 = tmp1 * fpm(a, n - 1) % p;
	tmp2 = tmp2 * fpm(b, n - 1) % p;
	printf("%lld\n", (tmp1 + tmp2) % p);
}

} // namespace STDquantum

int main() {
	STDquantum::main();
	return 0;
}
```

# value

## 题目描述

给定 $n$ 个物品，每个物品的价值为 $$v_i$$,代价为$$w_i$$。 可以以任意的顺序选择任意数量的物品，但在选择编号为 $i$ 的物品后，剩下的物品的价值都会减少$$w_i$$。要求最大化选择商品的价值之和。

## 输入输出格式

### 输入格式

第一行包含两个整数 $n$。 之后$n$行每行包含两个整数$$v_i,w_i$$。

输出格式

共一行包含一个整数，表示价值之和的最大值。

## 输入输出样例

### 输入样例 #1

```
5
8 2
10 7
5 1
11 8
13 3
```

### 输出样例 #1

```
27
```

## 说明

对于$40\%$ 的数据：$n \leq 8$。 

对于$100\%$的数据：$$n \leq 5000,v_i,w_i \leq 10^5 $$。

## 做法

可以看出来这题是类似DP的一种东西，可以先考虑令$f_{i,j}$表示已经考虑到了第$i$个物品，已经选了$j$个能得到的最大价值。

然后我们考虑怎么来写DP方程。然后写一写就会发现用贪心思想尽可能把$$w_i$$小的往里面装，根本写不下去，因为你并不知道里面已经选了什么物体，它们的$$w_i$$分别是多少，再添加下一个物品时，就会涉及到很多问题。所以怎么解决呢？

既然放最小的会影响后面决策，那么不妨从大的开始放。可以发现在放最后一个物体的时候对背包里的物体是无影响的，而在放倒数第二个的时候，要减掉的也只是背包内物体个数乘上倒数第二个物体的代价。

于是另一种贪心放置的思路有了。就先从大到小排个序，然后枚举往里丢就可以了。

转移：
$$
f[i][j]=\max(f[i-1][j],\ f[i-1][j-1]-(j-1)*a[i].w+a[i].v)
$$
前半部分就是不选这个物品，后半部分就是选了这个物品。

当然可以发现$n$的范围是$[1,\ 5000]$，对于空间限制$n^2$有点大了。所以滚动数组就可以。具体的嘛，开$f[0/1][n]$，来回的区分就用$i\&1$就可以，相邻两个异或$1$是可以互换的，相邻两个数不会一样，也能方便地转换。

代码如下：

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

const int N = 10086, inf = -0x7fffffff;
int n, ans = inf, f[2][N];
struct M {
	int v, w;
} a[N];
bool cmp(M a, M b) {
	if (a.w == b.w) {
		return a.v > b.v;
	} else {
		return a.w > b.w;
	}
}
inline void main() {
	scanf("%d", &n);
	for(int i = 1; i <= n; ++i) {
		scanf("%d%d", &a[i].v, &a[i].w);
	}
	sort(a + 1, a + n + 1, cmp);
	for(int i = 1; i <= n; ++i) {
		for(int j = 1; j <= i; ++j) {
			f[i & 1][j] = max(f[(i & 1) ^ 1][j], f[(i & 1) ^ 1][j - 1] - (j - 1) * a[i].w + a[i].v);
		}
	}
	for(int i = 0; i <= n; ++i) {
		ans = max(ans, f[n & 1][i]);
	}
	printf("%d", ans);
}

} // namespace STDquantum

int main() {
	STDquantum::main();
	return 0;
}
```

# binary

## 题目描述

给定一个长度为 $n$ 的整数数列 $a$ 和 $q$ 次操作： 修改操作：形如 $1$ $x$ $y$,表示将$$a_x$$ 的值修改为 $y$； 询问操作：形如 $2$ $x$ $y$，表示询问$$\sum_{i=1}^n(a_i + x)$$ $and$ $y$的值。

## 输入输出格式

### 输入格式

第一行包含两个整数 $n,q$。 第二行包含 $n$ 个整数 $a_i$。 之后 $q$ 行每行包含三个整数 $opt,x,y$。

输出格式

对于每一个询问操作，输出一行包含一个整数，表示询问操作的答案。

## 输入输出样例

### 输入样例 #1

```
6 6
8 9 1 13 9 3
1 4 5
2 6 9
1 3 7
2 7 7
1 6 1
2 11 13
```

### 输出样例 #1

```
45
19
21
```

## 说明

对于$40\%$的数据：$n,q \leq 5000$。 

对于另外 $20\%$ 的数据：所有的询问操作满足 $x =0$。 

对于$100\%$的数据：$n,q\leq 100000,0\leq a_i,x,y < 2^{20}$。

## 做法

因为要做与运算，所以考虑 $$a_i+x$$ 对 $y$ 每二进制位的贡献。

把 $y$ 拆分成不同位考虑，则只有当 $y$ 的第 $j$ 位是$1$时，且 $$a_i+x$$ 该位也为$1$时，一个$$a_i$$对答案有 $1<<j$ 的贡献，所以考虑如何求出该位为$1$的 $$a_i+x$$ 数量。

由于只考虑该位，则对应区间$D$为$[1<<j,\ 1<<(j+1)-1]$。在该区间内的数第 $j$ 位必定为$1$。

可以考虑用树状数组完成单点查询、区间查询操作，维护右端点为 $1<<(j+1)$ 的区间内符合要求的数。因为不用考虑更高位，则可以用 $$a_i$$ 模 $1<<(j+1)$ 再加上一，即取值在 $[1,\ 1<<(j+1)]$ 内。

可能的 $j$ 取值有$20$个，对应 $x,y <2^{20}$，即开二十个树状数组即可。

在查询时，树状数组求和类似前缀和。还需要注意最初求的区间$D$为 $a_i+x$ 的范围，则需要将整个区间向左平移 $x$ 个单位才是 $$a_i$$ 的实际范围，令该平移后区间为$M$，然而由于是在取模意义下操作，平移后可能会出现右端点在维护区间内，而左端点的值小于一的情况（左端点取模后会比$M$区间右端点更接近$D$区间右端点的情况），那么所求解的范围为图中$$R_1$$点左侧与$$L_1$$点右侧。

代码如下

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

const int N = 2e5 + 10;
int n, q, a[N], t[23][(1 << 20) + 3];
long long ans;
inline int lowbit(int x) {
	return x & (-x);
}
inline void update(int x, int k, int pos) {
	while (x <= (1 << pos + 1)) {
		t[pos][x] += k;
		x += lowbit(x);
	}
}
inline int query(int x, int pos) {
	int sum = 0;
	while (x) {
		sum += t[pos][x];
		x -= lowbit(x);
	}
	return sum;
}
inline void main() {
	scanf("%d%d", &n, &q);
	for(int i = 1; i <= n; ++i) {
		scanf("%d", a + i);
		for(int j = 0; j <= 19; ++j) {
			update(a[i] % (1 << j + 1) + 1, 1, j);
		}
	}
	for(int i = 1, op, x, y; i <= q; ++i) {
		scanf("%d%d%d", &op, &x, &y);
		if (op == 1) {
			for(int j = 0; j <= 19; ++j) {
				update(a[x] % (1 << j + 1) + 1, -1, j);
				update(y % (1 << j + 1) + 1, 1, j);
			}
			a[x] = y;
		} else {
			ans = 0;
			for(int j = 0, l, r; j <= 19; ++j) {
				if ((y & (1 << j)) == 0) continue;
				l = ((1 << j) - 1 - x + (1 << 20)) % (1 << j + 1);
				r = ((1 << j + 1) - 1 - x + (1 << 20)) % (1 << j + 1);
				if (l <= r) {
					ans += 1ll * (1 << j) * (query(r + 1, j) - query(l + 1, j));
				} else {
					ans += 1ll * (1 << j) * (query(r + 1, j) - query(l + 1, j) + query((1 << j + 1) + 1, j));
				}
			}
			printf("%lld\n", ans);
		}
	}
}

}

int main() {
	STDquantum::main();
	return 0;
}
```
