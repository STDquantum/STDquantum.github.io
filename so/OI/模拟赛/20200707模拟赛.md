# 单峰

## 题目背景

## 题目描述

小 $X$ 归来后，首先对数列很感兴趣。他想起有一类特殊的数列叫单峰数列。

我们说一个数列 {$a_i$} 是单峰的，当且仅当存在一个位置 $k$ 使得

$$
a_i < a_{i+1}(i<k)
$$

且

$$
a_i>a_{i+1}(i\ge k)
$$

现在小$X$ 想知道，对于 $1$ 到 $n$ 的所有排列，其中有多少个是单峰数列。

## 输入输出格式

### 输入格式

第一行包含一个整数 $n$。

### 输出格式

第一行包含一个整数，表示答案除以$10^9 + 7$ 的余数。

## 输入输出样例

### 输入样例 #1

```
2
```

### 输出样例 #1

```
2
```

## 说明

对于$20\%$ 的数据，$n \leq10$。 

对于$50\%$ 的数据，$n\leq10^5$。 

对于$100\%$ 的数据，$2\leq n \leq 10^{18}$。

## 做法

可以发现，由于是$n$的全排列，所以$n$为单峰的峰顶，可以考虑$n$在序列中的位置。由于左右两边是严格递增递减的，所以$n$的左右两侧数字固定，其可能的排列只有一种。所以考虑如何分配左右两边的数字。假设$n$在第二位，那么$n$左侧的数字为任意非$n$数字，即有$n-1$种可能，右边为递减序列，对总数不造成影响。假设$n$在第三位，那么$n$左侧的数字为任意两个非$n$数字，即从$n-1$个数字里挑出$2$个，就是组合数$C_{n-1}^2$。以此类推，可以得到最终的答案：
$$
\sum_{i=0}^{n-1}\left(\begin{array}c{n-1}\\{i}\end{array}\right) \bmod\ 1000000007
$$
这个式子的值应该记住，就是$2^{n-1}$，再用快速幂来做就好了。

~~其实直接打表也能发现规律（小声~~

代码如下：

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

const long long p = 1e9 + 7;
long long fpm(long long a, long long b) {
    long long ans = 1;
    for(; b; b >>= 1, a = a * a % p) {
        if (b & 1) ans = ans * a % p;
    }
    return ans;
}
long long n;
inline void main() {
    scanf("%lld", &n);
    printf("%lld\n", fpm(2, n - 1));
}

} // namespace STDquantum

int main() {
    STDquantum::main();
    return 0;
}
```

# 积木

## 题目描述

小$X$ 感到很无聊，从柜子里翻出了自己小时候玩的积木。 这套积木一共有 $n$ 块，每块积木都是一个长方体。小$X$ 想用这些积木拼成一个积木塔（不必每一块积木都使用）。 所谓积木塔，就是将积木一个一个摞起来，（除去最底层的积木外）每块积木的底面必须能被它下面的积木的底面完全包含（即对应的长宽都要更小或相等）。当然，积木可以任意放置，即可以以任意一面作为底面。 现在小$X$ 想知道，积木塔最大能拼多高。

## 输入输出格式

### 输入格式

第一行包含一个整数$n$。 接下来 $n$ 行，每行包含三个整数$a, b,c$，表示该块积木是一个$a \times b \times c$ 的长方体。

### 输出格式

第一行包含一个整数，表示答案。

## 输入输出样例

### 输入样例 #1

```
3
8 7 6
3 9 4
1 10 5
```

### 输出样例 #1

```
18
```

## 说明

### 样例解释 

选择第1块积木和第3块积木。 

### 数据范围 

对于$10\%$的数据，$n = 1$。 

对于$40\%$的数据，$n \leq 6$。 

对于$100\%$的数据，$1 \leq n\leq 15,1\leq a,b,c \leq 10^8$。

## 做法

设$f[s][i][0/1/2]$为最高高度，$s$为已选块集合，$i$是最顶上的块，$0/1/2$代表$i$朝上的面。

枚举下一个块如何放上去即可。

记得特判，或者加入一个无穷大的块。

代码：

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

const int N = 16;
int n, a[N], b[N], c[N], f[(1 << N) + 5][N][3], ans;
inline void main() {
    scanf("%d", &n);
    for(int i = 1, t[3]; i <= n; ++i) {
        scanf("%d%d%d", &t[0], &t[1], &t[2]);
        sort(t, t + 3);
        a[i] = t[0], b[i] = t[1], c[i] = t[2];
    }
    if (n == 1) {
        printf("%d", max(a[1], max(b[1], c[1])));
        return;
    }
    for(int i = 1; i <= n; ++i) {
        f[1 << i - 1][i][0] = c[i];
        f[1 << i - 1][i][1] = a[i];
        f[1 << i - 1][i][2] = b[i];
    }
    for(int x = 0, s; x < (1 << n); ++x) {
        for(int i = 1; i <= n; ++i) {
            for(int j = 1; j <= n; ++j) {
                if (!(x & (1 << i - 1)) || (x & (1 << j - 1))) continue;
                s = x | (1 << j - 1);
                if (a[i] >= a[j] && b[i] >= b[j]) {
                    f[s][j][0] = max(f[s][j][0], f[x][i][0] + c[j]);
                } // 0-0
                if (a[i] >= b[j] && b[i] >= c[j]) {
                    f[s][j][1] = max(f[s][j][1], f[x][i][0] + a[j]);
                } // 1-0
                if (a[i] >= a[j] && b[i] >= c[j]) {
                    f[s][j][2] = max(f[s][j][2], f[x][i][0] + b[j]);
                } // 2-0
                if (b[i] >= a[j] && c[i] >= b[j]) {
                    f[s][j][0] = max(f[s][j][0], f[x][i][1] + c[j]);
                } // 0-1
                if (b[i] >= b[j] && c[i] >= c[j]) {
                    f[s][j][1] = max(f[s][j][1], f[x][i][1] + a[j]);
                } // 1-1
                if (b[i] >= a[j] && c[i] >= c[j]) {
                    f[s][j][2] = max(f[s][j][2], f[x][i][1] + b[j]);
                } // 2-1
                if (a[i] >= a[j] && c[i] >= b[j]) {
                    f[s][j][0] = max(f[s][j][0], f[x][i][2] + c[j]);
                } // 0-2
                if (a[i] >= b[j] && c[i] >= c[j]) {
                    f[s][j][1] = max(f[s][j][1], f[x][i][2] + a[j]);
                } // 1-2
                if (a[i] >= a[j] && c[i] >= c[j]) {
                    f[s][j][2] = max(f[s][j][2], f[x][i][2] + b[j]);
                } // 2-2
                ans = max(ans, max(f[s][j][0], max(f[s][j][1], f[s][j][2])));
            }
        }
    }
    printf("%d", ans);
}

} // namespace STDquantum

int main() {
    STDquantum::main();
    return 0;
}
```

# 同余

## 题目描述

小$X$望着草稿纸上的数列，结合自己对同余的粗浅认识，又想到了一个新问题。 对于一个长度为 $n$ 的数列 {$a_i$}，每次询问将给出一组数$l, r, p,q$，小$X$ 想知道有多少个$i$ 满足$l \leq i \leq r$ 且 $a_i \equiv q\ (\bmod p)$。 小$X$ 沉迷这个问题无法自拔，因此一共进行了$m$ 次询问。

## 输入输出格式

### 输入格式

第一行包含两个整数 $n,m$。 第二行包含$n$ 个整数，表示数列{$a_i$}。 接下来$m$ 行，每行包含四个整数$l, r, p,q$，表示一次询问。

### 输出格式

$m$ 行，每行包含一个整数，表示该次询问的答案。

## 输入输出样例

### 输入样例 #1

```
5 2
1 5 2 3 7
1 3 2 1
2 5 3 0
```

### 输出样例 #1

```
2
1
```

## 说明

对于$20\%$ 的数据，$a_i \leq1$。 

对于$60\%$ 的数据，$a_i \leq100$。 

对于$100\%$ 的数据，$1 \leq m \leq 10^5，1 \leq l \leq r \leq n \leq 10^5，0 \leq q < p \leq 10000，0 \leq a_i \leq 10000$。

## 做法

分块做一下前缀和就可以了qwq

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

const int N = 2e5 + 10, M = 10003;
int n, m, a[N], f1[317][M], f2[317][M], d, maxx, ans;
inline void main() {
    scanf("%d%d", &n, &m);
    d = sqrt(n);
    for(int i = 1; i <= n; ++i) {
        scanf("%d", a + i);
        ++f1[i / d][a[i]];
        maxx = max(maxx, a[i]);
    }
    for(int i = 0; i <= maxx; ++i) {
        f2[0][i] = f1[0][i];
    }
    for(int i = 1; i <= d + 1; ++i) {
        for(int j = 0; j <= maxx; ++j) {
            f2[i][j] = f2[i - 1][j] + f1[i][j];
        }
    }
    for(int i = 1, l, r, p, q, ll, rr; i <= m; ++i) {
        scanf("%d%d%d%d", &l ,&r, &p, &q);
        ans = 0;
        if (l / d > r / d - 2) {
            for(int j = l; j <= r; ++j) {
                ans += (a[j] % p == q);
            }
            printf("%d\n", ans);
        } else if (p == 1 && q == 0) {
            printf("%d\n", r - l + 1);
        } else {
            ll = l / d + 1, rr = r / d - 1;
            for(int j = q; j <= maxx; j += p) {
                ans += f2[rr][j] - f2[ll - 1][j];
            }
            if (l % d == 0) {
                l = l / d;
                for(int j = q; j <= maxx; j += p) {
                    ans += f1[l][j];
                }
            } else {
                for(int j = l; j % d; ++j) {
                    ans += (a[j] % p == q);
                }
            }
            if (r % d == d - 1) {
                r = r / d;
                for(int j = q; j <= maxx; j += p) {
                    ans += f1[r][j];
                }
            } else {
                for(int j = r - r % d; j <= r; ++j) {
                    if (!j) continue;
                    ans += (a[j] % p == q);
                }
            }
            printf("%d\n", ans);
        }
    }
}

} // namespace STDquantum

int main() {
    STDquantum::main();
    return 0;
}
```
