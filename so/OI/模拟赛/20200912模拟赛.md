# 小学组(xx)

## 题目背景

对于确定的一个大小为 $n$ 的数组 $A$，数组中每个元素是一个 $m$ 维向量，同时有一个向量 $x$，定义一个操作顺序 $p$ 是“好”，当且仅当其满足如下条件： 

（1）序列中每个数均满足 $1\le p_i\le n $

（2）$i\neq j\ \Leftrightarrow\ p_i\neq p_j$ 

（3）$\forall i,A_{p_1,i}\oplus A_{p_2,i}\oplus\cdots\oplus A_{p_k,i}=x_i$

其中 $k(1\le k\le n)$ 为 $p$ 的长度，$\oplus$ 为一种位运算(&是与)(|是或)(^是异或)。

## 题目描述

现在给定 $A$ 和 $x$ 以及 $\oplus$，请你求出有多少组满足条件的 $p$。这个答案可能很大，请你求出对 $10^9+9$ 取模的结果。

## 输入输出格式

### 输入格式

第 $1$ 行一个字符 $\oplus$ 和两个整数 $n,m$ 如题目中描述。

第 $2$ 到 $n + 1$ 行每行 $m$ 个数表示A

第 $n + 2$ 行 $m$ 个数表示 $x$

### 输出格式

输出一个数对应答案。

```
1 1 0 0 
```

### 输出样例 #3


## 说明

Subtask 1(10pts)： $n,m\le 7$。

Subtask 2 (10pts)： $n,m\le 10$。

Subtask 3 (5pts)：$\oplus=$`&`。

Subtask 4 (5pts)：$\oplus=$`|`。

Subtask 5 (5pts)：$\oplus=$`^`。

Subtask 6 (20pts)：$n,m\le 15$。

Subtask 7 (30pts)： $n,m\le 20$。

Subtask 8 (15pts)：无特殊限制。

对于全部数据，有$1\le n,m\le 25$， $A_{i,j}\in\{0,1\}$。

## 题解

发现这三个运算符都有结合律和交换律，所以如果求出一个序列可行，那么随机打乱也可行。所以只需要找到是哪几个结合了就好。

#### 法一

枚举子集，然后把子集内部这些都运算一下。

#### 法二

那这个常数巨大的dfs怎么做呢？

所以我们需要做的是把前面的操作记住。这样再计算就省下了很多时间。

所以怎么记住呢？

考虑开一个二维数组，第一位表示之前已经计算过的向量个数，第二位表示是向量里第几个数（其实就相当于开了很多个记录历史操作的向量）。

枚举要计算的向量个数，然后再枚举一个起始点，开始dfs，dfs里记录开始枚举的位置和已经计算过的长度。

每当新加进来一个向量时，就直接在当前长度减一的那个历史版本上修改。

因为是dfs，得到第一个答案后回溯，然后接着枚举填哪个，所以正确性是没有问题的。

至于之前已经填过这一位的情况，由于是直接在历史版本上修改，所以这里面的数对将要操作完成的那个向量也没有影响。

之前卡了三十多分钟的常数，跑 $n=m=25$ 的数据从 $6$ 秒多卡到了 $1$ 秒多，这开了O2的话就稳了。不过看来评测机速度还可以，没开O2也能过（luogu过不了）。

#### 卡常

卡常神器：bitset！

由于向量之间都是统一进行同一操作，所有元素都是 $0/1$，所以bitset有得天独厚的优势（还有 $\displaystyle\frac 1{32}$ 的小常数）。

实际上由于最长只有25，所以直接用一个 int 类型存就完了，好像是快很多。真是学傻了。

看代码吧

```cpp
#include <bits/stdc++.h>
using namespace std;

namespace STDquantum {

template <typename T> inline void read(T &x) {
  char ch;
  while (ch = getchar(), !isdigit(ch)) {}
  x = (ch ^ 48);
}

typedef long long ll;
const int N = 50, mod = 1e9 + 9; // 注意mod不是1e9+7，虽然这也是个质数

int fac[N];
inline void initFac(int n) { // 预处理阶乘
  fac[0] = 1;
  for (int i = 1; i <= n; ++i) fac[i] = fac[i - 1] * (ll)i % mod;
}

int n, m, lenMax;
bitset<N> X, A[N], tmp[N]; // A是
ll ans;
char x;
void dfs(int begin, int lenNow) {
  if (lenNow > lenMax) {
    if (tmp[lenMax] != X) goto school;
    (ans += fac[lenMax]) %= mod;
  school:
    return;
  }
  if (lenNow == 1) {
    tmp[1] = A[begin];
    dfs(begin + 1, 2);
  } else {
    for (int i = begin; i <= n && n - i + lenNow >= lenMax; ++i) {
      /* 开始位置的枚举加了一个范围判定，如果把i连后面全取了也不够lenMax个就不要枚举了 */
      if (x == '&') {
        tmp[lenNow] = tmp[lenNow - 1] & A[i];
      } else if (x == '|') {
        tmp[lenNow] = tmp[lenNow - 1] | A[i];
      } else {
        tmp[lenNow] = tmp[lenNow - 1] ^ A[i];
      }
      dfs(i + 1, lenNow + 1);
    }
  }
}
inline void main() {
  x = getchar(), scanf("%d%d", &n, &m);
  initFac(n);
  for (int i = 1, op; i <= n; ++i) {
    for (register int j = 1; j <= m; ++j) {
      read(op);
      A[i][j] = (op > 0);
    }
  }
  for (int i = 1, op; i <= m; ++i) read(op), X[i] = (op > 0);
  for (lenMax = 1; lenMax <= n; ++lenMax) {
    for (register int i = 1; i <= n && n - i + 1 >= lenMax; ++i) {
      dfs(i, 1);
    }
  }
  printf("%lld", ans);
}

}  // namespace STDquantum

int main() {

  freopen("D:\\OI\\STDquantum\\test.out", "w", stdout);
#endif
#ifndef LOCAL
  freopen("xx.in", "r", stdin);
  freopen("xx.out", "w", stdout);
#endif
#endif
  STDquantum::main();
  return 0;
}
```
