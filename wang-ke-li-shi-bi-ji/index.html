中国古代商业发展的基本历程

  1. 原始社会末期：以物易物的原始商业产生

  2. 商：出现职业商人和最早的货币

    古代商族始祖契的六世孙王亥很会做生意，经常率领很多奴隶，贩运货物到远处去做买卖。以后，做买卖成了商族人的主要谋生手段之一。

    周朝建立后，商族人地位每况愈下，于是纷纷重操旧业，到处去跑买卖。周族人鄙视生意人，在其心目中，做买卖的人就是商人，这就是中国称生意人为“商人”的缘由。“商业”、“商品”、“商旅”诸词也都由此而来。

  3. 西周： “工商食官” ，流通骨贝、 铜贝

  4. 春秋战国：货币种类繁多；“工商食官” 制度被打破， 私商成为商人主体；出现巨商大贾和繁荣的商业中心

    • 春秋战国时著名的商业中心：齐：临淄，赵：邯郸，魏：大梁，楚：郢、宛

  5. 秦统一货币，以 “半两钱“ 通行天下

  6. 汉：商运活跃，对外贸易发展：海、陆“丝绸之路” （陆上为主）；汉武帝将铸币权收归国有：五铢钱（我国钱币史上使用时间最长的货币）

  7. 唐：水、 陆商运发达（“俞大娘航船” ）；客舍、邸店等辅助性经营场所生意兴隆；出现柜坊和飞钱。

    “大历、贞元间，有俞大娘航船最大，居者养生、送死、嫁娶悉在其间，开巷为圃，操驾之工数百，南至江西，北至淮南，岁一往来，其利甚溥。”

    ​ ——《唐国史补》卷下

    1. 邸店：亦称“邸舍”、“邸铺”等，唐代以后供客商堆货、交易、寓居的行栈的旧称。
    2. 柜坊：又称傲柜等。从邸店中衍生出的专为富商大贾、贵族官僚等人储存、保管金银财物的商户。我国最早的银行雏形，是唐代商品经济发展的反映。跟现在银行给存款人付利息不同，财务存放者需要向柜坊缴纳租金。柜坊使商人不必携带大量的铜钱，方便了远途贸易。
    3. 飞钱：又称“便换”，是中国历史上早期的汇兑业务形式。本身不介入流通，不行使货币职能，因此不是真正意义上的纸币。有两种形式：一是官办，商人在京城把钱交给诸地设于京城的“进奏院”，由进奏院开具一张凭证，记载着地方及钱币数目，商人携券到所去地区指定地方取钱；二是私办，大商人在各道或主要城市有联号或交易往来，代营“堡垫”,以此牟利。“飞钱”减轻了商人携带大量钱币外出经商的不便，同时减少了市场对铜钱的需求，可以缓解当时钱币的不足的问题。

  8. 宋元：海外商运发达，海上丝绸之路，海外贸易税成为财政收入的重要来源。北宋， 益州出现世界上最早的纸币 “交子” 。

    北宋时期的工商税一度占70%（其中一半来自海上贸易），而农业税只占30%。

    北宋时，由于铸钱的铜料紧缺，政府为弥补铜钱不足，在一些地区大量地铸造铁钱，但铁钱笨重不便。伴随着造纸和印刷技术的进步，纸币交子在四川地区应运而生。起初是由商人发行（商业信用凭证：存款凭证$\to$纸币），后改由政府发行，成为法定货币(“官交子”)。交子的出现，是我国古代货币史上由金属货币向纸币的一次重要演变。交子是我国最早的纸币，也是世界上最早的纸币。其流通范围基本限于四川境内，后来虽在陕西、河东有所流行，但不久就废止了。南宋纸币：会子。

    • 思考：宋元时对外贸易形式逐渐以海运取代陆运的原因？

      一艘商船， 可载六七万斤的货物， 相当于2000只骆驼的运载量。

      1. 航运成本较低、运量大
      2. 路上丝绸之路因战乱时断时续，而海运受地形及战乱的影响较小
      3. 造船、航海技术的提高（宋代指南针的应用）
      4. 经济重心南移

  9. 明清：

    • 区域间长途贩运贸易发展较快，出现商帮和会馆（晋商、徽商）。

      商帮：区域性商人群体。

      材料

      明中期以后，由于商业竞争的加剧，一些商人为壮大队伍、扩大资本、增强竞争实力，便以地域为中心、以血缘乡谊为纽带，自发组成了既“亲密”又松散的商人群体。他们借助群体的力量，操纵某些地区、某些行业的商业贸易，从而赢得高额利润。

      会馆：随着商帮出现而产生。同一地域商人在异地所建的停宿、储运、交际的重要场所。

    • 明代白银成为流通货币，银、钱并用，还有纸币（宝钞）；清代出现票号。

      票号：票庄、汇兑庄，是一种专门经营汇兑业务的金融机构，兴盛于清朝。晋商首创，中国第一家票号日升昌。

      日升昌，创建于道光四年（公元1824年），座落于大清金融第一街平遥古城大街的繁华地段，分号遍布全国30余个城市、商埠重镇，远及美、东南亚等国，以“汇通天下”著称于世。结束了我国镖局押送现银的落后金融局面，极大地加速了商业运转和货币流通，掀开了中国金融史的光辉一页。存放款、放贷等。

“市” 的发展（交易场所）

  1. 第一阶段：市坊制，管理严格，西周—唐代

    • 特征：

      • 集中贸易；
      • 严格的时、空限制；（县治以上城市设市，市、坊分开）
      • 政府直接监管（设官）。

    • 秦：严格管理

      材料

      “有买及卖也，各婴其价，一物不能名一钱者，勿婴”。“为作务及官府市，受钱必辄入钱缿中，令市者见入，不从令者赀一甲”。

      商品买卖必须明码标价。 从事手工业和为官府出售产品的人， 收受金钱时必须当面把钱投入陶制容钱器之中， 违反法令者要受到处罚。

    • 汉：长安九市；设“市署”（机构）、“市令”（官员）管理

    • 魏晋南北朝：出现草市，并设官管理（草市尉）

      • 特点：

        • 离城镇较远；
        • 交通便利地区；
        • 自然形成的民间集市；
        • 也受到政府商业机构的管理。

      • 意义：

        • 是对官办贸易的有益补充；
        • 推动了中国古代商品经济的发展。

    • 唐朝

      1. 市坊制：市、坊分开，“市”有严格的时空限制；商品交易市场专业化明显

      2. 变化：夜市繁荣。

        草市成为相对集中的地方商业中心$\to$市坊制不再适应商品经济发展的需要

    • 宋朝：“市”突破时间和地点的限制

      “太祖乾德三年(965)四月十三日，诏开封府，令京城夜市自三鼓以来，不得禁止。”

      ——《宋会要辑稿》

  2. 第二阶段：市坊制被彻底打破，放宽限制，宋代—明清

    • 宋朝

      • 市坊制被彻底打破，“市”突破时空限制；
      • 原先不允许设“市”的城郊和农村，也出现了“市”；
      • 商业活动不再受官吏的直接监管；
      • “草市”已具备比较完备的饮食服务设施；
      • 夜市、晓市繁荣

    • 明清：都市商业区已相当繁华

    • 总结

      • 时间：由限时到不限时。
      • 空间：突破限制，地域范围拓展一从城市到乡村，从城市中固定的官市到分散于街巷。
      • 国家管理：从严格限制到不再受官吏直接监管。
      • 功能：从单纯的商品买卖市场发展为服务设施比较完备、功能齐全的商业区。
      • 交易种类日益多元。

商业都会的崛起

总结：古代中国城市发展的规律

  • 功能：政治功能渐弱，经济功能渐强。唐代以前的城市多是政治中心、军事重镇，发展过程中兼具经济职能；唐代以来，城市多以工商业发展而兴起。

  • 从商业管理看：政府逐渐放宽对城市商业活动的限制。

  • 数量不断增多，规模不断扩大；明清时期还兴起了一大批工商业市镇。
  • 地域分布：从北向南，从内地向沿海。唐朝以前主要分布在北方，唐以后，伴随着经济重心南移，南方都市数量明显多于北方。明清时期，新兴工商业市镇大都分布在江南地区。
  • 发展类型：商品化、专业化趋势加强。明朝中后期江南某些地区出现资本主义萌芽。

古代货币发展演变的规律

  1. 类型：由自然货币向人工货币演变（商周）。
  2. 形制：由形状杂乱多样向规范统一演变（秦）。
  3. 铸币权：由地方铸币向中央铸币演变（汉武帝），体现中央集权趋势。
  4. 材质：由金属货币向纸币转化（北宋交子）。
  5. 命名方式：由重量货币向名称货币（元宝、通宝）演变（唐高祖“开元通宝”)。
  6. 铸造工艺：由人工铸币向机制货币演变（清末）。

古代中国商业经济发展的特点

  1. 起源早，发展水平不断提高。
  2. 国内贸易和海外贸易全面繁荣。明清时期海外贸易由盛转衰。
  3. 商业交通发达：水、陆商运兴盛，长途贩运发达。
  4. 商人群体活跃（明清：商帮）。
  5. 商业市场形式多样（市、草市、夜市、晓市）。
  6. 货币种类不断丰富。
  7. 城市商品经济发达，明清时期兴起一批专业化工商业市镇。城市经济职能不断增强。
  8. 地位低下。受重农抑商政策压制，商人社会地位较低；始终是农耕经济的补充而未占据古代经济的主导。

四、古代中国的经济政策

  • 课标要求：了解“重农抑商”、“海禁”等政策及其影响，分析中国资本主义萌芽发展缓慢的原因。

重农抑商政策

  1. 确立：秦国商鞅变法

  2. 含义：（重本弃末、强本弱末）

    • 以农业为本业，工商业为末业

  3. 原因

    • 根本原因：自然经济下社会生产力水平低下的必然产物

    • 农业的重要性：农业是国民经济的基础，可提供最基本的生活资料；为封建国家提供稳定的财政来源；将农民束缚在土地上，有利于社会稳定，维护封建统治。

    • 工商业的缺陷：不能提供最基本的生活资料；其发展又会加剧劳动力从土地上流失，造成种种社会问题。（影响农业发展、不利于中央集权、市场稳定等）
    • 法家（重农抑商）及儒家思想（重义轻利）的影响

  4. 主要表现、手段

    1. 重农：

      • 强调农业的重耍性，以农业为本业；
      • 采取—系列措施促进农业发展：鼓励垦荒、抑制土地兼并、分给农民一定数量土地（均田制）、奖励耕织、轻徭薄赋、兴修水利、制定历法、抑制工商业发展保证农业劳动力等
      • 以身示范：藉田礼
      • 编纂农书，注重总结农业生产经验。

    2. 抑制私营工商业

      • 强调农业为本，工商业为末，构建轻视商人的主流价值取向。
      • 贬低商人地位（限制生活方式、四民之末，市籍、不得为官）
      • 征收重税

      • 加强管理，限制商人经营范国（立法，对内：市坊制度；对外：海禁和闭关锁国政策）

      • 发展官营手工业和商业（盐铁官营），与民争利。加强经济控制

      • 强化户籍管理，限制人口流动

  5. 历代抑商政策的具体表现

  6. 重农抑商政策发展的阶段特点

  7. 评价重农抑商政策

  8. 政策与现实之间的差距

    • 抑商与商业的发展、商人的活跃

    • 农本思想、抑商主张与“重商”思想（典型：黄宗羲“工商皆本”）

    • 社会价值观：“贱商”传统与经商求富的逐利观念、拜金主义观念、崇奢风气，等级观念的淡化
    • 抑制私营手工业与私营手工业的不断发展：明中叶以后，制瓷、矿冶、纺织等诸行业的民营手工业逐渐超过官营手工业，占据手工业的主导。

资本主义萌芽的产生与缓慢发展

  1. 概念

  2. 标志/实质：雇佣劳动关系

  3. 产生条件

  4. 根本原因：社会生产力和商品经济的发展

  5. 产生时间、地区

  6. 资本主义萌芽发展的特点

  7. 明清资本主义萌芽缓慢发展的原因

    • 劳动力缺乏：自然经济占主导，农民被束缚在土地上。
    • 市场狭小：小农经济自给自足，且农民贫困，购买力低，国内市场狭小；明清海禁、闭关锁国政策严格限制海外贸易，国外市场无法打开。

    • 资金缺乏：政府抑商征收重税；商人经商致富后，往往将资金用于买田置地而非扩大再生产，缺乏资本积累。

    • 政府政策阻碍：重农抑商政策、“海禁”和闭关锁国政策。

    • 根本原因：腐朽落后的封建制度。

  8. 意义：革命性和进步性

    • 是在中国封建社会内部产生的新的生产关系，冲击传统的经济形态（封建自然经济），推动反封建专制的早期“民主”思想产生。

    • 代表近代化的曙光，标志着封建制度趋向衰落。

专题二：近代中国资本主义的曲折发展

第一课时

鸦片战争后中国经济结构的变动

表现

  • 自然经济开始解体

  • 农产品商品化程度加深

  • 洋行的大量出现

  • 买办阶层的产生

  • 贸易中心转移

原因：外国资本主义的入侵

实质：中国被迫卷入资本主义世界市场，成为西方世界市场的经济附庸。➔沦为外国资本主义的商品市场和原料产地。

影响：灾难与机遇并存

  • 中国被强行纳入资本主义世界市场体系，成为西方资本主义的经济附庸；
  • 自然经济逐渐解体，动摇了封建统治的经济基础；
  • 客观上促进了商品经济的发展，为资本主义经济发展准备了市场和劳动力条件，促进了近代民族工业的兴起。

近代民族工业的起步

  1. 洋务企业——第一批近代民族工业

  2. 民族资本主义企业（私人创办的近代民族工业）

    产生原因：

    • 自然经济解体，为资本主义发展提供劳动力、商品市场、资金、原料等条件。
    • 洋务运动的诱导。
    • 外资企业的利润刺激。

    时间：十九世纪六七十年代

    途径：部分中小地主、官僚、买办、商人、华侨等投资于近代工矿企业；部分手工工场采用机器生产，转变为近代工矿企业。

民族资本主义的初步发展（甲午战后）

  1. 原因

    ①列强加紧资本输出，中国的自然经济进一步解体；

    ②在内外交困形势下，清政府放宽对民间办厂的限制；

    ③在严重民族危机刺激下，出现“实业救国”思潮。

  2. 表现：企业数量增加、规模扩大

  3. 企业代表：

  5. 影响

    ①政治：民族资产阶级作为新的政治力量，开始登上历史舞台（维新派、革命派），为民主革命的发展提供了经济和阶级基础。

    ②思想：有利于资产阶级维新思想和革命思想的产生和传播，有利于思想解放。

民族资本主义发展“短暂的春天”（1912-1919）

  1. 原因

    ①政治环境：辛亥革命推翻专制统治，扫清障碍；民国建立提高政治地位，发展实业法令，激发热情；

    ②社会环境：群众性反帝爱国运动（抵日货、倡国货）

    ③“实业救国”，民族资产阶级爱国心

    ④外部条件：一战期间，欧洲列强忙于战争，暂时放松了对中国的经济侵略

  2. 表现

    ①1912一1919八年间，新建厂矿企业、新增资本发展速度和规模空前，超过过去半世纪成就。

    ②纺织业是当时最大的新式工业。

    ③面粉业是第二大工业。扩大内销，大量出口。

    ④火柴、榨油等轻工业显著发展，重工业有一定增长。

  3. 特点：

    ①主要分布在沿海地区；

    ②发展主要在轻工业方面（纺织、面粉业），重工业基础薄弱，未形成独立完整的工业体系；

    ③一些主要工业部门中，外国资本仍然超过民族资本；

    ④虽有一定发展，但自然经济仍占绝对优势；

    ⑤发展速度快，但时间短暂。

国民党统治后期民族资工业发展的困境（1937-1949）

原因：

  1. 1937-1945日本全面侵华战争；

  2. 抗战结束后，国民党发动内战、恶性通货膨胀政策、繁重的捐税；

  3. 美国的经济侵略；

  4. 官僚资本的压迫；

影响中国民族资本主义发展的因素

  1. 推动因素
    • 西方列强的侵略客观上为民族资本主义的兴起和发展创造了条件（分解自然经济、改变生产方式和思想观念、放松侵略）
    • 政府鼓励兴办实业 ——政策
    • 人民反帝爱国运动的推动 ——社会基础
    • 民族资产阶级自强不息的爱国精神 ——源泉和动力
  2. 阻碍因素
    • 诞生和发展在半殖民地半封建社会，深受帝国主义、封建主义和官僚资本主义的压迫和束缚 ——最主要的阻碍因素
    • 自身弱点：先天不足（缺乏资本、人才、技术、市场和思想观念的准备）、后天畸形（地域分布、行业发展不均）
    • 近代中国长期政局动荡，缺乏稳定社会环境

民族资本主义的命运

  • 在夹缝中求生存

    • 外国资本主义阻碍民族资本主义发展

  • 在斗争中求发展

    • 原因

      1. 深受三座大山压迫（斗争对象）
      2. 民族资本主义发展，民族资产阶级力量壮大（经济基础、阶级基础）
      3. 为维护本阶级利益和中华民族利益（斗争目的）

    • 斗争方式

      • 变法、革命
      • 罢市、游行抗议
      • 兴办实业、抵制洋货
      • 捐款捐物，支援抗战

    • 主要事例

    • 斗争特点：民族资产阶级的两面性

      • 革命性
      • 妥协性

    • 斗争结局

      反帝反封建不彻底，不可能领导中国人民取得民主革命的胜利。

民族资本主义在中国近代史上的地位和作用

  1. 经济：是一种新的经济因素，冲击、瓦解着封建自然经济，促进中国经济的近代化，对外国资本主义的经济入侵也起了抵制作用。
  2. 政治：使民族资阶产生和发展壮大，为维新变法及辛亥革命等资阶民主革命提供社会基础；壮大了无阶力量，为新民主主义革命及中共建立准备了阶级条件。
  3. 思想：冲击和动摇了封建思想的统治地位，推动了西方资本主义思想文化的传播。
  4. 发展趋势：先天不足➔后天畸形，一定程度上依赖三座大山难以独立发展，产生很大消极作用（民族资阶的软弱性和妥协性，不可能领导中国民主革命取得胜利）。
